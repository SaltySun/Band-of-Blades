# 刀锋连队 — 核心机制手册

> 版本：v1.2 拆分版 | 用途：所有玩家和GM的基础参考 | 页数：精简

---

## 一、战役总览与核心调性

### 一句话设定
刀锋军团是一支溃败的雇佣兵团，在西方不死者大军的追击下向东撤退，目标是穿越阿尔德马克到达天刃堡进行最后的决战。

### 核心调性
- **黑暗军事奇幻**：文艺复兴晚期科技水平（火绳枪、黑火药、炼金术）
- **没有救世主**：没有精灵矮人，没有救世主魔法
- **只有鲜血、泥坑、漫长的行军，和必死的觉悟**

### 主题
这款游戏的核心不是"打赢战争"，而是**"在注定失败的撤退中，决定你愿意为什么牺牲，以及你能守护住多少人性"**。

### 关键数字
- 完整战役约 **12节**（12次任务+战役循环）
- 到达天刃堡 = 最终防守战 = 战役结束

## 二、核心骰子机制

### 基本投骰
- 投若干 **d6**，取**最大值**作为结果
- **6** = 完全成功
- **4-5** = 部分成功（做到了，但有代价/后果）
- **1-3** = 失败（状况变糟）
- **两个以上6** = 关键成功（额外优势）
- 骰池为0或负数时，投**两个骰子取较小值**（无法关键成功）

### 五种骰子类型
| 骰子类型 | 用途 | 投骰时机 |
|---------|------|---------|
| **行动骰** | PC进行有挑战性的行动 | 玩家声明行动时 |
| **抵抗骰** | 减轻或避免后果 | 玩家不想接受后果时 |
| **遭遇骰** | 任务开始时确定处境 | 任务部署完成后，GM投 |
| **战役骰** | 战役阶段的职位行动 | 战役阶段各职位执行行动时 |
| **幸运骰** | GM判断不确定事件 | GM需要随机决定时 |

### 11种通用行动
| 属性 | 行动 | 适用场景 |
|------|------|---------|
| **认知** | 调查、射击、修补、社交、侦查、训诫 | 信息收集、远程战斗、设备维护、谈判、侦察、纪律 |
| **技术** | 游击、破坏、机动、组织 | 近战骚扰、摧毁设施、移动/驾驶、指挥调度 |
| **决心** | 动摇 | 抵抗恐惧、坚守阵地、鼓舞士气 |

### 奖励骰来源（最多+2d）
1. **协助**（其他玩家角色协助）：+1d
2. **超越自我**：+1d 或 +1效果等级（消耗2点压力）
3. **恶魔的交易**：其他玩家或GM提出代价，换取+1d

## 三、行动骰六步法

当玩家宣布要进行有挑战性的行动时，按以下步骤执行：

```
1. 玩家宣布目标
2. 玩家选择行动类型（11种通用行动之一）
3. GM设定处境（安全/危险/绝望）
4. GM设定效果等级（有限1格/一般2格/极佳3格/极限4格）
5. 玩家添加奖励骰（协助+1d / 超越自我+1d / 恶魔的交易+1d，最多+2d）
6. 投骰，判断结果和后果
```

### 处境设定
| 处境 | 判断标准 | 失败后果 | 特殊奖励 |
|------|---------|---------|---------|
| **安全** | 玩家有压倒性优势 | 小麻烦/效果降低 | — |
| **危险（默认）** | 势均力敌 | 伤害/麻烦/处境恶化 | — |
| **绝望** | 玩家处于明显劣势 | 重伤/严重麻烦/死亡风险 | 对应属性+1经验 |

### 效果三要素
GM判断效果等级时考虑：
- **效力**：针对弱点、额外时间、更大风险（二元：有效/无效）
- **规模**：敌人数量、区域大小（规模优势可恶化处境、降低效果、提升伤害）
- **威胁**：装备质量/生物危险度（普通=1，优质=2，重武器=3+）

| 效果等级 | 格子数 | 含义 |
|---------|--------|------|
| **极限** | 4+ | 超乎想象的好 |
| **极佳** | 3 | 额外的好成果 |
| **一般** | 2 | 正常预期成果 |
| **有限** | 1 | 部分或较弱结果 |
| **无效** | 0 | 毫无效果 |

### 用处境交换效果
玩家可主动将处境降级（危险→绝望）以换取效果提升（一般→极佳）。

## 四、闪回机制

闪回允许玩家在任务进行中**临时插入一个之前发生过的场景**，执行一个行动来解决当前困境。

### 闪回消耗
- 简单闪回（"我之前藏了一把刀"）= **1点压力**
- 复杂闪回（"我早上侦查了巡逻路线"）= **2点压力**

### 闪回限制
- **不能**改变已经确定的事实
- **不能**闪回到"不可能"的时间点
- 闪回中的行动**仍需投骰**

### 闪回常见用途
| 用途 | 范例 | 压力 |
|------|------|------|
| 提前侦查 | "闪回到今天早上，我侦查了巡逻路线" | 2点 |
| 设置陷阱 | "闪回到出发前，我在门后设置了绊索" | 1-2点 |
| 准备物品 | "闪回到昨天，我偷了看守的钥匙" | 1点 |
| 建立关系 | "闪回到上周，我贿赂了城门守卫" | 2点 |

## 五、团队协作系统

### 1. 协助（Assist）
- **消耗**：**1点精神压力**
- **效果**：被协助者 **+1骰**
- **限制**：每次行动骰只能接受1位协助
- **风险**：协助者可能承受后果

### 2. 领导集体行动（Lead Group Action）
- **场景**：多人做同一件事（全队冲锋、集体侦察）
- **机制**：领导者投骰；参与者各消耗**1点压力**+**+1骰**（最多+3d）
- **风险**：领导者失败则**所有参与者承受后果**

### 3. 准备行动（Setup Action）
- **效果**：接下来配合你的角色**改善1级处境**或**提升1级效果**
- **范例**：狙击手压制敌人火力 → 队友机动从危险→安全

### 4. 保护（Protect）
- **场景**：队友承受后果时挺身而出
- **消耗**：**1-2点压力**
- **效果**：原目标不受后果，保护者承受全部后果

## 六、抵抗、压力与创伤

### 抵抗骰机制
当玩家不想接受后果时：
1. 选择对应属性（认知/技术/决心）投抵抗骰
2. 承受 **6 - 骰子结果** 点**精神压力**

### 精神压力系统
- **上限**：6点（可通过"幸存者"能力提升至8/10）
- **用途1**：抵抗后果
- **用途2**：超越自我（2点压力 = +1骰 或 +1效果等级 或 失能状态下行动）

#### 压力阈值效果
| 压力值 | 状态 |
|--------|------|
| **0-2点** | 正常 |
| **3-4点** | 感到紧张，描述应反映焦虑 |
| **5点** | 濒临崩溃，可能出现轻微失误 |
| **6点** | **精神创伤** |

### 精神创伤
- 压力满6点时 → 受到**精神创伤**（失能状态）
- 选一个**创伤症状**（8种之一）
- 最多2次创伤 = **死亡**（可通过"老兵"能力提升至3/4次）
- 扮演创伤 → 获得经验点

### 8种创伤症状
冷漠、阴影、痴迷、偏执、鲁莽、软弱、无常、恶毒

## 七、伤害、腐化与死亡

### 伤害等级
| 等级 | 名称 | 范例 | 效果 |
|-----|------|------|------|
| 1级 | 轻微 | 挫伤、擦伤、扭伤 | 轻度妨碍 |
| 2级 | 中度 | 流血、烧伤、疲劳 | 明显妨碍 |
| 3级 | 严重 | 断腿、穿胸、昏厥 | **失能** |
| 4级 | 致命 | 致命毒素、胸部下陷 | **=死亡** |

- **行满溢推**：2级满→升为3级；3级满→升为4级（死亡）
- **护甲抵消**：普通护甲1格，重甲2格，盾牌1格

### 腐化与枯萎病
- 累计 **7点腐化** → 获得1个枯萎症状，枯萎病+1级
- 4级枯萎病 = **变成怪物 = 死亡**
- 腐化**无法**通过一般手段降低
- 不死者造成的腐化通常等于其威胁度

### 8种枯萎症状
咒诅、寄生、饥渴、瘴气、变异、狂怒、腐烂、幻象

## 八、进程表系统

| 类型 | 用途 | 填满触发 |
|-----|------|---------|
| **危机表** | 敌人疑心增长、仪式进行 | 危机爆发 |
| **竞速表** | 两个对立表同时推进 | 谁先满谁赢 |
| **连锁表** | 条件解锁型 | 前置表满后启动 |
| **任务表** | 时间窗口倒计时 | 时间到 |
| **威胁表** | 精英/著名怪物血量 | 怪物被击败 |
| **长期项目** | 军需官大型工程 | 项目完成 |

**填表规则**：
- 有限效果 → 1格 / 一般效果 → 2格 / 极佳效果 → 3格
- 威胁表：威胁度2→8格，威胁度3→10格

## 九、威胁度与敌人

### 威胁度尺度

| 威胁度 | 对应对象 |
|--------|---------|
| **1** | 普通军团兵、新兵、腐尸 |
| **2** | 精英不死者（暗影女巫、无心者、黑栎骑士） |
| **3** | 臭名昭著的怖物（哀嚎者、奇美拉、杂碎包） |
| **4** | 副官（"博士"、"狼人"比希德） |
| **5+** | **仅限**破碎者、余烬之王、神选者 |

### 威胁度影响

| 影响项 | 规则 |
|--------|------|
| **效果等级** | 敌人威胁度每比你高1级 → 你的效果等级降1级 |
| **伤害等级** | 敌人威胁度/规模每比你高1级 → 你受到的伤害提升1级 |
| **腐化点数** | 不死者造成的腐化通常等于其威胁度 |
| **经验奖励** | 每次任务XP = 遭遇的最高敌人威胁度 |

### 规模等级

| 规模 | 描述 |
|------|------|
| 1 | 一两个个体 |
| 2 | 小型队伍（3-5人） |
| 3 | 中型队伍（6-10人） |
| 4 | 大型队伍（20人） |
| 5 | 巨型队伍（40人） |
| 6 | 超巨型队伍（80人） |
| 7 | 极巨型队伍（160人） |

### 敌人图鉴

| 敌人 | 威胁度 | 特性 |
|------|--------|------|
| **腐尸** | 1 | 最基本的不死者，数量庞大 |
| **痰尸** | 2 | 吐酸性浓液，腐化=威胁度 |
| **暗影女巫** | 2 | 诅咒和精神攻击，威胁表8格 |
| **无心者** | 2 | 不知疲倦的战斗机器 |
| **黑栎骑士** | 2 | 重甲不死骑兵 |
| **杂碎包** | 3 | 膨胀炸弹，威胁表10格 |
| **哀嚎者** | 3 | 缝合巨型怪物，4米+ |
| **奇美拉** | 3 | 兽化人怖物，多种动物脑袋 |
| **"博士"** | 4 | 凋零者手下，精于缝制怖物 |
| **"狼人"比希德** | 4 | 兽化人副官 |
| **"发条刺客"卢戈斯** | 4 | 乌鸦副官，护甲免疫黑弹 |
