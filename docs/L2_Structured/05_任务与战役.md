# 任务与战役

> **用途**：任务类型、游戏结构、结算流程、营地场景的完整规则。
> **来源**：`L1/刀锋连队_战役管理手册.md` + `L1/刀锋连队_AI-GM手册_v1.3.md`

## 1. 游戏结构

```
┌─────────────────────────────────────────────────┐
│                    战役阶段                         │
│  ├── 时间流逝（积累时间和压力，消耗食物）            │
│  ├── 各职位执行战役行动                             │
│  │   ├── 军需官：获取资源/长期项目/休息恢复/募兵     │
│  │   ├── 间谍总管：派遣间谍（短期/长期任务）          │
│  │   └── 书记官：讲述军团故事                        │
│  ├── 指挥官决定是否进军                             │
│  └── GM生成新任务 → 进入任务阶段                     │
└──────────────────────┬────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────┐
│                    任务阶段                         │
│  ├── 指挥官选主要任务+次要任务                      │
│  ├── 军士长派人选 + 军需官分配补给                   │
│  ├── 投遭遇骰确定起始处境                            │
│  ├── 主要任务：小队执行，投骰、承担后果              │
│  │   └── 可使用闪回、团队协作                        │
│  ├── 次要任务：由遭遇骰决定结果                      │
│  ├── "在营地"短暂扮演                                │
│  └── 结算：记录奖惩、士气损失 → 回到战役阶段         │
└─────────────────────────────────────────────────┘
```

- **一节游戏** = 一次任务阶段 + 一次战役阶段
- **完整战役**约 12节
- 到达天刃堡 = 最终防守战 = 战役结束

## 2. 任务类型系统

### 标准任务类型

每个地点有一组标准任务，指挥官从中选择主要和次要任务。

| 任务类型 | 描述 | 典型奖励 | 典型惩罚 | 适合专家 |
|---------|------|---------|---------|---------|
| **突袭任务** | 快速打击敌人目标，制造混乱，摧毁设施 | 士气+1，消灭敌人，削弱敌军 | 伤亡，敌人警觉提升，报复 | 重装兵、狙击手、军官 |
| **侦察任务** | 侦查敌情、地形、资源，获取情报 | 情报+1，了解威胁度，发现弱点 | 被发现，遭遇战，情报泄露 | 斥候、军官 |
| **宗教任务** | 与神选者相关的精神/信仰任务，净化、祈祷、守护圣物 | 神选者支持，士气+1，净化腐化 | 亵渎，腐化+，神选者不满 | 书记官、医疗兵 |
| **补给任务** | 获取食物、物资、装备，搜刮废墟，交易 | 食物+，补给+，获得特殊物品 | 物资损失，遭遇伏击，食物污染 | 军需官、军官 |

### 特殊任务
- 每个地点有**3个特殊任务**
- 通常有独特的背景、奖励和惩罚
- 需要消耗**1份情报**，由**GM选择**特殊任务并告知指挥官
- 完成后可能改变战役走向

### 次要任务
- 由指挥官选择的第二个任务
- **不需要玩家扮演执行**，由遭遇骰结果自动决定成败
- 成功 = 获得奖励；失败 = 承受惩罚
- **所有参与了次要任务的专家获得2点xp，不论任务成功与否**
- 可以用情报为次要任务的遭遇骰+1
- 次要任务可以和主要任务派不同的人

### 次要任务投骰规则
- 使用**次要任务参与者的调查骰池**投遭遇骰
- 结果对应地点的次要任务奖励/惩罚表：
  - **1** = 惩罚A（通常是士气-1或物资损失）
  - **2** = 惩罚B
  - **3** = 惩罚C
  - **4** = 奖励A（通常是情报或物资）
  - **5** = 奖励B
  - **6** = 奖励C（通常是士气+1或稀有物资）

### 任务选择流程

```
1. 指挥官查看当前地点的所有可用任务
2. 使用情报提问（了解威胁、敌人、奖励）
3. 选择主要任务（需玩家扮演执行）
4. 选择次要任务（自动结算）
5. 军士长部署人员
6. 投遭遇骰确定初始处境
7. 执行任务
```

## 3. 结算阶段流程

结算阶段是任务阶段结束后、战役阶段开始前的关键环节，按以下顺序执行：

```
1. 主要任务结算
   ├── 存活角色获得任务奖励
   └── 死亡角色从名册移除
2. 次要任务结算
   ├── 投遭遇骰（使用次要任务参与者的调查骰池）
   └── 根据结果分配奖励/惩罚
3. 伤害与压力处理
   ├── 角色勾销伤害（根据休息条件）
   ├── 处理压力满6点的角色（获得创伤）
   └── 处理4级伤害的角色（死亡）
4. 士气调整
   ├── 阵亡：士气-1
   ├── 士气6+：结算时+1士气
   └── 士气3-：结算时-1士气
5. 经验点分配
   ├── 基础XP = 本次最高敌人威胁度
   └── 额外XP（协助/扮演创伤/承受伤害/死亡等）
6. 晋升判定
   └── 新兵存活2次任务 → 晋升为士兵
7. 进入"在营地"场景
```

### 主要任务结算
- 所有参与主要任务且**存活**的角色获得任务奖励
- **死亡**的角色从名册中移除（保留在编年史中）
- 每个死亡角色使**士气-1**

### 伤害与压力处理
- **压力满6点**：清空压力→获得精神创伤→选择症状→失能
- **4级伤害**：角色死亡
- 如果同时满足压力满和4级伤害，先处理死亡

### 晋升判定
- 每个新兵检查**存活任务次数**
- 达到**2次存活** → 晋升为士兵
- 晋升时：获得忍耐特种行动1级，获得1个新能力

## 4. 在营地场景

"在营地"是每次任务结束后、战役阶段开始前的叙事场景。根据当前**士气水平**，场景氛围截然不同。

### 士气0-3：沮丧营地

**氛围**：沉默、易怒、绝望、阴影笼罩

**机制效果**：
- 书记官讲故事时，听众很难获得经验点
- 休息时只能勾销**1格伤害**（而非正常的2格）
- **士气3或更低**：结算时额外-1士气
- 可能出现**逃兵事件**

**AI-GM叙事建议**：
- 描述潮湿、寒冷、腐臭味
- 让玩家角色感受到"我们可能要死在这里"

### 士气4-7：普通营地

**氛围**：疲惫但仍在坚持、例行公事、麻木

**机制效果**：
- 书记官讲故事正常结算经验点
- 休息时可以正常勾销伤害和清除压力

### 士气8-10：振奋营地

**氛围**：士气高涨、歌声、笑声、充满希望

**机制效果**：
- 书记官讲故事时，所有参与者额外+1经验点
- 休息时额外勾销**1格伤害**
- **士气6+**：结算时额外+1士气
- 可能出现**自愿加入的新兵**

### 营地活动

| 活动 | 效果 | 限制 |
|------|------|------|
| **休息** | 勾销伤害，清除压力 | 需要安全营地和食物 |
| **修理装备** | 修补行动恢复装备 | 需要工具和材料 |
| **交换情报** | 社交行动分享信息 | 无限制 |
| **祷告/冥想** | 动摇行动恢复精神 | 需要安静环境 |
| **训练新兵** | 训诫行动帮助新兵 | 需要老兵参与 |
| **写遗书** | 无机制效果，纯叙事 | 任何角色 |
