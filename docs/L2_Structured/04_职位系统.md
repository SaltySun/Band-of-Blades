# 职位系统

> **用途**：5个军团职位的完整规则，包括战役阶段行动和资源管理。
> **来源**：`L1/刀锋连队_职位系统手册.md` + `L1/刀锋连队_AI-GM手册_v1.3.md`

---

## 1. 指挥官（Commander）

### 核心职责
- 选择主要任务与次要任务
- 决定进军路线（选择下一个地点）
- 决定是否进军（推进战役）
- **消费情报**获得战略优势

### 核心资源：情报

**情报来源**：任务奖励（主要是侦察任务）

**情报消费方式**：
1. **遭遇骰+1**（消耗1点情报）：为任何任务增加1个遭遇骰
2. **解锁特殊任务**（消耗情报）：获得执行某地特殊任务的权利
3. **情报问题**（不消耗，提问数量 = 持有的情报数量上限）

### 情报问题列表（按持有量分级）

**0情报**（眼前任务细节）：
- 任务中不死者威胁度最高分别是多少？
- 任务需要多远距离行军？
- 哪件物品带上比较有用？
- 任务中有哪两种可选执行方式？
- 哪些小队不尊重或不信任领导层？
- 神选者对任务的态度是怎样的？

**1情报**（额外问题）：
- 任务中哪些物品比较有用？
- 谁对这个任务很认真？
- 可能遇到什么样的挑战？
- 会遇上哪些破碎者的部队？
- 接下来地点能找到什么有益资源？
- 某个精英的弱点/欲望可能是什么？

**2情报**（战略级）：
- 会遇到哪个副官或臭名昭著的存在？
- 不死者布置中最大的弱点是什么？
- 这个任务和之前的任务有关联吗？
- 破碎者在往哪个方向移动？
- 地点存在什么样的挑战或困难？

**3情报**（战役级）：
- 此前遇到的某个臭名昭著的存在的深层弱点是什么？
- 余烬之王的目标是什么？
- 破碎者如何影响当前地区的？
- 不死者军队正在执行什么计划？
- 军团如何才能最大限度地利用当前局势？

### 按神选者的初始情报
- **施芮亚**：每完成2次主要任务获得1份情报
- **有角者**：初始情报1
- **左拉**：初始情报0

## 2. 军士长（Sergeant）

### 核心职责
- **部署人员**：选择哪些角色参与主要/次要任务
- **管理士气**：士气追踪（0-10）
- **投遭遇骰**：决定任务起始处境

### 士气系统

- 士气 **0-10** 追踪
- 低士气 = 惩罚/叛逃风险
- 成功任务、宗教任务提升士气

#### 士气调整

| 条件 | 士气变化 |
|------|---------|
| 有角色阵亡 | -1 |
| 士气当前≥6 | +1（庆祝胜利） |
| 士气当前≤3 | -1（恐慌蔓延） |
| 完成宗教任务 | +1 |
| 完成突袭任务 | +1 |

### 遭遇骰

- **1-3** = 绝望处境
- **4-5** = 危险处境
- **6** = 安全处境

### 派遣人员原则
- 考虑角色的状态（失能、创伤、高腐化不宜派遣）
- 考虑角色的专业能力匹配任务类型
- 一个角色不能同时参与主要和次要任务
- 死亡角色无法派遣

## 3. 书记官（Lorekeeper）

### 核心职责
- 记录**编年史**（军团的故事与传说）
- 在"在营地"阶段**讲述军团故事**
- 提供叙事框架与背景信息

### 编年史类型（5种）

1. **军团建立的故事**：军团最初如何组建
2. **军团独立的故事**：军团何时成为独立雇佣兵
3. **战火淬炼的故事**：某场关键战役的传说
4. **意志不屈的故事**：绝境中坚守
5. **军团存在的意义**：军团的信念与使命

### 故事讲述规则
- 每次战役阶段，书记官可以讲述一个故事
- 参与玩家可获得对应属性的**经验点**
- 士气8-10时，所有听众额外+1 XP

## 4. 军需官（Quartermaster）

### 核心职责
- 物资调配与补给
- 募兵（补充伤亡）
- 休息与恢复（管理角色状态）
- 长期项目（制造攻城武器、研发药物等）

### 战役行动（每次战役阶段选一项）

#### 1. 获取资源
- 军需官投战役骰，结果决定获得多少食物/补给
- 可通过渠道、勒索等获得加成

#### 2. 放假（R&R）
- 让军团休整，士气+1
- 每个角色可以勾销伤害格子
- 但时间和压力会推进

#### 3. 长期项目
- 推进大型工程进程表
- 范例：制造攻城武器、研发新药、修理装备
- 通常需要多个战役阶段完成

#### 4. 募兵
- 补充新兵
- 新兵数量取决于声誉和士气
- 新兵默认无特种行动，威胁度1

### 休息与恢复详细规则

#### 6小时的休息
- 勾销**1格伤害**（任意等级）
- 清除部分压力（2-3点）
- 营地不安全时效果减半

#### 1天的休息（完整休整）
- 勾销**2格伤害**（任意等级）
- **清空所有压力**（归零）
- 新兵晋升判定

#### 休息前提条件
- **必须有足够食物**：每个存活角色消耗1单位食物
- **营地必须相对安全**
- 食物不足时：无法有效休息，士气下降，可能出现饥饿状态

### 食物消耗与补给系统

#### 食物消耗
- 每次战役阶段（进军后），每个存活角色消耗**1单位食物**
- 初始食物：**10单位**

#### 补给获取途径
1. **补给任务**：完成补给类型任务获得食物
2. **军需官"获取资源"**：投战役骰获得物资
3. **渠道（社交行动）**：利用关系获取额外补给
4. **间谍网络"商人"节点**：强化补给任务额外+1补给
5. **地点搜刮**：某些地点有特殊资源

## 5. 间谍总管（Spymaster）

### 起始配置
- **2位初始间谍**：1位熟练 + 1位大师
- 最多同时拥有 **3位间谍**（需解锁"增员"节点）

### 7位可选间谍

| 间谍 | 特技 | 定位 |
|------|------|------|
| **安托瓦内特** | 起始即为**大师级** | 暗杀/破坏 |
| **鲍提斯** | 扩张网络任务额外+1格 | 导师/建设 |
| **赤烦风** | **任何任务都不会负伤** | 万能安全牌 |
| **伊葛丽德** | 收集情报时**额外多问1个问题** | 情报专家 |
| **莉娅** | 调查长期任务+1骰 | 侦察/潜伏 |
| **昂也廷** | 强化任务长期任务+1骰 | 任务优化 |

### 间谍网络科技树

```
间谍网络（起始）——允许维持2位间谍
├── 增员 → 可同时维持3位间谍
│   └── 训练 → 熟练间谍升为大师（可点2次）
├── 分析师 → 调查长期任务+1骰
│   └── 投资 → 扩张网络长期任务+1骰
├── 现场评估 → 收集情报时额外多问1个问题
├── 诱捕 → 设置陷阱长期任务+1骰
└── 信息源 → 强化任务长期任务+1骰
    ├── 商人 → 强化补给任务额外+1补给
    ├── 教团 → 强化宗教任务额外+1资源
    ├── 佣兵 → 强化突袭任务额外+1士气
    └── 游侠 → 强化侦察任务额外+1情报
```

### 派遣间谍（与军需官同期）

每个战役阶段，一名间谍只能执行**短期任务**或**长期任务**之一。

#### 短期任务（无风险，不投骰）

| 任务 | 效果 |
|------|------|
| **恢复** | 移除间谍的负伤状态 |
| **收集情报** | 从指挥官情报问题列表中选1个问题问GM |
| **勒索** | 为1次"获取资源"战役行动+1骰 |
| **协助** | 为1次"长期项目"战役行动+1骰 |

#### 长期任务（8格进程表，有伤亡风险）

**投骰规则**：
- 熟练间谍：1d
- 大师间谍：2d（取最高）
- 1-3 = 1格（**负伤**）
- 4-5 = 2格
- 6 = 3格
- 双6 = 5格

**伤亡规则**：
- 首次负伤 → 获得负伤状态
- 已负伤再次负伤 → **牺牲（死亡）**
- 负伤间谍执行"恢复"短期任务可清除负伤

| 长期任务 | 完成效果 |
|---------|---------|
| **强化任务** | 指定类型任务奖励与惩罚骰各+1 |
| **扩张网络** | 解锁一个新网络节点 |
| **设置陷阱** | 下轮任务列表必定包含突袭指定副官/怖物的任务 |
| **招募** | 加入1位新间谍（替换死亡者） |
| **调查** | 了解当地所有特殊任务；指挥官已解锁权限时由间谍总管选择执行哪个 |

> 长期任务可接力：不同间谍轮流推进同一任务，暂停无惩罚。
