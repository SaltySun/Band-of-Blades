## 1. 三选一

### 施芮亚（巴尔塔治愈女神）
- **主题**：军事与战略
- **初始威胁度**：4
- **初始情报**：每完成2次主要任务获得1份情报（来自有角者外出探索观察）

### 有角者（潘雅丛林之神）
- **主题**：神秘力量与变形
- **初始情报**：1份（带回敌人部队配置和地形信息）

### 左拉（泽姆亚远古神明）
- **主题**：强力正面作战
- **初始情报**：0（军团没时间侦察）

## 2. 神眷形态

神选者拥有多种形态/能力，按战役进度解锁：

### 施芮亚进化路线
阿斯莉卡之慈悲/佑/泪 → 受膏者 → 战斗圣女（**神选者之血**，血液成为圣物） → 战争圣徒

### 有角者进化路线
赏/眼/筋 → 变形者 → 受膏者 → 伟大猎手/森林之翼/白腿马之皮

### 左拉进化路线
晨星 → 圣印 → 永生神之怒/吻/力

---

## 3. 神选者战役规则

- 神选者不能简单死去（可牺牲神眷等级避免死亡）
- 每次任务只能使用一次神选者能力
- 神选者不受腐化影响
- 神选者可以执行**超凡使命**（单人高难度任务）

---

## 4. 超凡使命

- 神选者可以执行**单人高难度任务**
- 不需要其他角色参与
- 通常涉及对抗破碎者的直接行动
- 失败后果严重（神选者受伤/被俘/神眷等级下降）
- 成功可大幅削弱破碎者或扭转战局
