# 角色系统

> **用途**：角色创建、升级、晋升、死亡的完整规则。
> **来源**：`L1/刀锋连队_特种士兵手册.md` + `L1/刀锋连队_AI-GM手册_v1.3.md`

---

## 1. 角色类型

| 类型 | 威胁度 | 特种行动 | 说明 |
|------|--------|---------|------|
| **新兵** | 1 | 无 | 未经考验的新人，存活2次任务晋升士兵 |
| **士兵** | 1 | 忍耐 | 饱经风霜的老兵，有多种能力可选 |
| **专家** | 1-2 | 职业专属 | 6种特种士兵：重装兵、医疗兵、军官、斥候、狙击手 |

> 专家统称"专家"，是军团的核心战斗力。新兵和士兵统称"小队成员"或"军团兵"。

---

## 2. 属性与行动等级

### 三大属性
- **认知**：调查、射击、修补、社交、侦查、训诫
- **技术**：游击、破坏、机动、组织
- **决心**：动摇

### 行动等级
每个行动有等级（0-4，通过能力可提升至4）：
- **0级（未受训）**：骰池为0，投2d取较小值
- **1级（受训）**：1d
- **2级（熟练）**：2d
- **3级（专家）**：3d
- **4级（大师）**：4d（需"精英"能力解锁）

### 初始行动等级
- **专家**：3个行动起始为1级，1个行动起始为2级，其余为0级
- **士兵/新兵**：大部分为0级，部分为1级

> 详见各职业的具体初始分配。

## 3. 经验与升级

### 获得XP的条件

| 条件 | XP奖励 |
|------|--------|
| 基础XP | = 本次任务最高敌人威胁度 |
| 高威胁击杀 | 狙击手射杀威胁度≥2 → +1 XP |
| 危险处境行动 | 在危险/绝望处境中行动 → 对应属性+1 XP |
| 协助队友 | 成功协助他人 → +1 XP |
| 角色扮演 | 营地场景有趣扮演（GM判断） → +1 XP |
| 体现创伤 | 扮演创伤症状制造困难 → +1 XP |
| 承受创伤 | 任务中承受精神创伤 → +1 XP |
| 承受重伤 | 任务中承受3级以上伤害 → +1 XP |
| 角色死亡 | 死亡角色遗赠 → +1 XP（分配给新角色） |
| 士气高涨 | 士气8-10时书记官讲故事 → 额外+1 XP |
| 新兵晋升 | 新兵晋升为士兵时 → +1 XP |

### 升级规则
- 填满某个行动的**经验表**（通常6格）→ 该行动等级+1
- 等级上限默认为**3级**
- 通过能力**"精英"**可提升至**4级**（需选择两次"精英"才能将两个行动提升至4级）

### 次要任务XP
- **所有参与了次要任务的专家获得2点XP，不论任务成功与否**

## 4. 晋升与死亡

### 新兵晋升
- 新兵存活**2次任务** → 晋升为士兵
- 晋升时：获得1个新能力，获得**忍耐**特种行动1级

### 角色死亡
- 4级伤害 = 死亡
- 第2次精神创伤（无"老兵"能力时）= 死亡
- 4级枯萎病 = 变成怪物 = 死亡

### 死亡处理
- 死亡角色从名册中移除（保留在编年史中）
- 每个死亡角色使**士气-1**
- 死亡角色遗赠**1点XP**给新角色

## 5. 角色创建流程

```
1. 选择角色类型（新兵/士兵/专家）
2. 如果是专家，选择特种士兵职业
3. 分配初始行动等级
4. 选择初始能力（专家选2项，士兵选1项，新兵选1项）
5. 确定初始压力（0）、伤害（无）、腐化（0）
6. 选择装备和负载等级
7. 命名并撰写简短背景
```

### 角色状态追踪

每个角色需记录：
- 姓名、类型、职业
- 11个行动的当前等级和经验表
- 压力值（0-6+）
- 伤害（4行，每行若干格）
- 腐化值（累计）
- 枯萎病等级（0-4）
- 创伤次数和症状
- 特种行动等级和剩余次数
- 装备和负载
- 存活任务次数（新兵用）
