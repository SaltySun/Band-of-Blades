# 核心机制

> **用途**：所有玩家和GM必须掌握的基础规则。
> **来源**：`L1/刀锋连队_AI-GM手册_v1.3.md` 第1-2章 + `L1/刀锋连队_核心机制手册.md`

## 1. 战役总览

### 一句话设定
刀锋军团是一支溃败的雇佣兵团，在西方不死者大军的追击下向东撤退，目标是穿越阿尔德马克到达天刃堡进行最后的决战。

### 核心调性
- **黑暗军事奇幻**：文艺复兴晚期科技水平（火绳枪、黑火药、炼金术）
- **没有救世主**：没有精灵矮人，没有救世主魔法
- **只有鲜血、泥坑、漫长的行军，和必死的觉悟**

### 主题
核心不是"打赢战争"，而是**"在注定失败的撤退中，决定你愿意为什么牺牲，以及你能守护住多少人性"**。

### 关键数字
- 完整战役约 **12节**（12次任务+战役循环）
- 到达天刃堡 = 最终防守战 = 战役结束
- 科技水平：**文艺复兴晚期**

## 2. 骰子系统

### 基本投骰
- 投若干 **d6**，取**最大值**作为结果
- **6** = 完全成功
- **4-5** = 部分成功（做到了，但有代价/后果）
- **1-3** = 失败（状况变糟）
- **两个以上6** = 关键成功（额外优势）
- 骰池为0或负数时，投**两个骰子取较小值**（无法关键成功）

### 五种骰子类型

| 骰子类型 | 用途 | 投骰时机 |
|---------|------|---------|
| **行动骰** | PC进行有挑战性的行动 | 玩家声明行动时 |
| **抵抗骰** | 减轻或避免后果 | 玩家不想接受后果时 |
| **遭遇骰** | 任务开始时确定处境 | 任务部署完成后，GM投 |
| **战役骰** | 战役阶段的职位行动 | 战役阶段各职位执行行动时 |
| **幸运骰** | GM判断不确定事件 | GM需要随机决定时 |

### 11种通用行动

| 属性 | 行动 | 适用场景 |
|------|------|---------|
| **认知** | 调查、射击、修补、社交、侦查、训诫 | 信息收集、远程战斗、设备维护、谈判、侦察、纪律 |
| **技术** | 游击、破坏、机动、组织 | 近战骚扰、摧毁设施、移动/驾驶、指挥调度 |
| **决心** | 动摇 | 抵抗恐惧、坚守阵地、鼓舞士气 |

### 奖励骰来源（最多+2d）

1. **协助**（其他玩家角色协助）：+1d
2. **超越自我**：+1d 或 +1效果等级（消耗2点压力）
3. **恶魔的交易**：其他玩家或GM提出代价，换取+1d

> ⚠️ 骰池上限：基础骰池 + 奖励骰最多+2d（即总骰池通常不超过5-6d）

## 3. 行动骰六步法

当玩家宣布要进行有挑战性的行动时，按以下步骤执行：

```
Step 1: 玩家宣布目标（要做什么 + 期望结果）
Step 2: 玩家选择行动类型（11种通用行动之一）
Step 3: GM设定处境（安全 / 危险 / 绝望）
Step 4: GM设定效果等级（无效0 / 有限1 / 一般2 / 极佳3 / 极限4+）
Step 5: 玩家添加奖励骰（协助+1d / 超越自我+1d / 恶魔的交易+1d）
Step 6: 投骰与判断
```

### Step 3: 处境设定

| 处境 | 判断标准 | 失败后果 | 特殊奖励 |
|------|---------|---------|---------|
| **安全** | 压倒性优势、敌人无防备 | 小麻烦/效果降低 | — |
| **危险（默认）** | 势均力敌 | 伤害/麻烦/处境恶化 | — |
| **绝望** | 明显劣势、敌人强大 | 重伤/严重麻烦/死亡风险 | 对应属性+1经验 |

> AI-GM提示：大部分情况用"危险"。只有明显优势才"安全"，明显劣势才"绝望"。

### Step 4: 效果等级

效果三要素：
- **效力**（二元）：针对弱点、额外时间、更大风险 → 有效/无效
- **规模**：敌人数量、区域大小
- **威胁**：装备质量/生物危险度（普通=1，优质=2，重武器=3+）

| 效果等级 | 格子数 | 含义 |
|---------|--------|------|
| **极限** | 4+ | 超乎想象的好 |
| **极佳** | 3 | 额外的好成果 |
| **一般** | 2 | 正常预期成果 |
| **有限** | 1 | 部分或较弱结果 |
| **无效** | 0 | 毫无效果 |

> 威胁度差距：敌人威胁度每比你高1级 → 效果降低1级

### 用处境交换效果
玩家可主动将处境降级（如危险→绝望）以换取效果提升（一般→极佳）。

### Step 6: 投骰结果

| 结果 | 含义 |
|------|------|
| **6** | 完全成功，按效果等级执行 |
| **4-5** | 部分成功，按效果等级执行但附加1个后果 |
| **1-3** | 失败，状况变糟，附加1-2个后果 |
| **两个6+** | 关键成功，额外获得优势 |

## 4. 抵抗、压力与创伤

### 抵抗骰机制
当玩家不想接受后果时：
1. 选择对应属性（认知/技术/决心）投抵抗骰
2. 承受 **6 - 骰子结果** 点**精神压力**

> 例：投出4 → 承受2点压力；投出1 → 承受5点压力

### 精神压力系统

- **默认上限**：6点
- **用途1**：抵抗后果
- **用途2**：超越自我（2点压力 = +1骰 或 +1效果等级 或 失能状态下行动）

#### 压力阈值效果

| 压力值 | 状态 |
|--------|------|
| **0-2点** | 正常，无明显影响 |
| **3-4点** | 感到紧张，部分行动描述应反映焦虑 |
| **5点** | 濒临崩溃，角色可能出现轻微失误、情绪失控 |
| **6点** | **精神创伤**（见下） |

#### 压力满6点时的详细流程

1. **清空所有压力标记**（回到0）
2. **获得一次精神创伤**
3. **选择一种创伤症状**（从8种中选择）
4. **角色进入失能状态**（除非使用能力如"不屈"）
5. **如果这是第2次创伤** → 角色**死亡**

#### 压力上限提升
- **"幸存者"**能力：上限6→8→10

#### 创伤承受上限提升
- **"老兵"**能力：上限2→3→4（第3/4次创伤才死亡）

### 8种创伤症状
冷漠、阴影、痴迷、偏执、鲁莽、软弱、无常、恶毒

> 扮演创伤症状制造困难 → 获得经验点

## 5. 伤害、腐化与死亡

### 伤害等级

| 等级 | 名称 | 范例 | 效果 |
|-----|------|------|------|
| 1级 | 轻微 | 挫伤、擦伤、扭伤、吓到 | 轻度妨碍 |
| 2级 | 中度 | 流血、烧伤、疲劳过度 | 明显妨碍 |
| 3级 | 严重 | 断腿、穿胸、昏厥 | **失能** |
| 4级 | 致命 | 致命毒素、胸部下陷 | **=死亡** |

- **行满溢推**：2级满→升为3级；3级满→升为4级（死亡）
- **护甲抵消**：普通护甲1格，重甲2格，盾牌1格

### 腐化与枯萎病

- 累计 **7点腐化** → 获得1个枯萎症状，枯萎病+1级
- 4级枯萎病 = **变成怪物 = 死亡**
- 腐化**无法**通过一般手段降低
- 不死者造成的腐化通常等于其威胁度

### 8种枯萎症状
咒诅、寄生、饥渴、瘴气、变异、狂怒、腐烂、幻象

## 6. 进程表系统

进程表是将复杂局势可视化的核心叙事工具，用扇形格子追踪进度。

### 进程表类型

| 类型 | 用途 | 填满触发 |
|-----|------|---------|
| **危机表** | 敌人疑心增长、仪式进行、气压升高 | 危机爆发 |
| **竞速表** | 两个对立表同时推进 | 谁先满谁赢 |
| **连锁表** | 条件解锁型 | 前置表满后启动 |
| **任务表** | 时间窗口倒计时 | 时间到 |
| **威胁表** | 精英/著名怪物血量 | 怪物被击败 |
| **长期项目** | 军需官大型工程 | 项目完成 |

### 填表规则

| 效果等级 | 格子数 |
|---------|--------|
| 有限效果 | 1格 |
| 一般效果 | 2格 |
| 极佳效果 | 3格 |

- 进程表代表**障碍**（如"不死者哨兵"），而非手段（如"躲过守卫"）
- 威胁表血量：威胁度2→8格，威胁度3→10格

## 7. 规模与威胁度

### 规模等级

| 规模等级 | 描述 |
|---------|------|
| 1 | 一两个个体 |
| 2 | 小型队伍（3-5人） |
| 3 | 中型队伍（6-10人） |
| 4 | 大型队伍（20人） |
| 5 | 巨型队伍（40人） |
| 6 | 超巨型队伍（80人） |
| 7 | 极巨型队伍（160人） |

规模差距影响：
- 规模每高1级 → 可恶化处境（危险→绝望）
- 规模每高1级 → 可降低效果1级
- 规模每高1级 → 可提升伤害1级

### 威胁度尺度

| 威胁度 | 对应对象 |
|--------|---------|
| **1** | 普通军团兵、新兵、腐尸、普通邪教徒、野生猛兽 |
| **2** | 优质装备专家、精英不死者（暗影女巫、无心者、黑栎骑士）、痰尸 |
| **3** | 臭名昭著的怖物（哀嚎者、奇美拉、杂碎包）、余烬卫士 |
| **4** | 副官级敌人（"博士"、"狼人"比希德）、攻城武器 |
| **5+** | **仅限**破碎者、余烬之王、神选者（战斗圣女形态） |

### 威胁度影响规则

| 影响项 | 规则 |
|--------|------|
| **效果等级** | 敌人威胁度每比你高1级 → 你的效果等级降1级 |
| **伤害等级** | 敌人威胁度/规模每比你高1级 → 你受到的伤害提升1级 |
| **腐化点数** | 不死者造成的腐化通常等于其威胁度 |
| **经验奖励** | 每次任务XP = 遭遇的最高敌人威胁度 |
