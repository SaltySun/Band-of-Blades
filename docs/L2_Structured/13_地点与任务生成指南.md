# 地点与任务生成指南

> **用途**：GM生成各地点的标准任务和特殊任务时的参考模板。
> **来源**：基于`L1/刀锋连队_战役管理手册.md`任务类型系统 + 战役世界观合理推导。
> **说明**：原始PDF因编码问题无法提取文本，各地点的具体任务列表无法直接从PDF复制。本文件提供**生成模板**和**示例地点**，GM可按模板快速生成任意地点的任务。

---

## 地点设计模板

每个地点应包含：

```
地点名称：
地点类型：[城市废墟/村庄/要塞/森林/沼泽/山脉/河流渡口]
环境特征：2-3个描述性关键词
撤退方向：[东/东南/东北]（向天刃堡方向）

标准任务（4个）：
1. 突袭任务：_____
2. 侦察任务：_____
3. 宗教任务：_____
4. 补给任务：_____

特殊任务（3个）：
A. _____（需1情报解锁）
B. _____（需1情报解锁）
C. _____（需1情报解锁）

次要任务奖励/惩罚表：
| 骰值 | 结果 |
|------|------|
| 1 | 惩罚A：_____ |
| 2 | 惩罚B：_____ |
| 3 | 惩罚C：_____ |
| 4 | 奖励A：_____ |
| 5 | 奖励B：_____ |
| 6 | 奖励C：_____ |

地点资源：_____（可搜刮的特殊物品/食物/情报）
地点威胁：_____（该地点常见敌人类型）
```

## 示例地点1：灰烬沼泽（Ashfen Marsh）

**地点类型**：沼泽
**环境特征**：毒雾、腐烂植被、木板栈道
**撤退方向**：东

### 标准任务

| 任务类型 | 任务名称 | 描述 | 典型奖励 | 典型惩罚 |
|---------|---------|------|---------|---------|
| **突袭** | 烧毁不死者孵化池 | 沼泽深处的腐化池中不断孕育新的不死者，潜入并摧毁它 | 士气+1，延缓敌军追击 | 伤亡，瘴气腐化+1 |
| **侦察** | 测绘安全通道 | 找到穿越沼泽的最快路线，标记危险区域 | 情报+1，下地点进军时间-1 | 陷入泥潭，物资损失 |
| **宗教** | 净化被亵渎的神龛 | 沼泽中有一座古老神龛被不死者亵渎，神选者要求净化 | 神选者支持，士气+1 | 亵渎反噬，腐化+2 |
| **补给** | 搜刮废弃渔村 | 沼泽边缘的渔村被遗弃，可能有食物和药品 | 食物+2，炼金试剂 | 食物污染，疾病 |

### 特殊任务

| 任务 | 描述 | 奖励 | 惩罚 |
|------|------|------|------|
| **A. 猎杀沼喉兽** | 一只巨大的变异怖物潜伏在沼泽深处，当地人称它为"沼喉" | 获得黑栎防具，士气+2 | 伤亡，恐惧蔓延 |
| **B. 营救被俘斥候** | 一名军团斥候在几天前侦察时被俘，被关押在沼泽中央的岛上 | 获得情报+2，救出专家 | 营救失败，士气-2 |
| **C. 打捞沉没物资** | 一艘沉没的补给船躺在沼泽底部，里面有军团急需的黑火药 | 黑弹+3，手雷+2 | 潜水危险，人员受伤 |

### 次要任务奖励/惩罚表

| 骰值 | 结果 |
|------|------|
| 1 | 惩罚A：士气-1，1人陷入沼泽失能 |
| 2 | 惩罚B：食物-1，瘴气导致1人腐化+1 |
| 3 | 惩罚C：物资损失，无奖励 |
| 4 | 奖励A：情报+1 |
| 5 | 奖励B：食物+1，炼金试剂 |
| 6 | 奖励C：士气+1，发现安全通道（下地点遭遇骰+1） |

### 地点资源
- 炼金试剂（化合软膏、枭目油）
- 沼泽草药（可制作解毒剂）
- 废弃渔网（军需官长期项目材料）

### 地点威胁
- 腐尸（大量，从泥中爬出）
- 痰尸（毒雾中潜伏）
- 瘴气区域（自然腐化来源）

## 示例地点2：黑栎要塞（Blackoak Keep）

**地点类型**：要塞废墟
**环境特征**：石墙、防御工事、制高点
**撤退方向**：东南

### 标准任务

| 任务类型 | 任务名称 | 描述 | 典型奖励 | 典型惩罚 |
|---------|---------|------|---------|---------|
| **突袭** | 夺回城墙控制权 | 要塞被不死者占领，从侧门潜入夺回城墙 | 士气+1，消灭大量敌人 | 伤亡，被围困 |
| **侦察** | 绘制敌军布防图 | 不死者在要塞内部重新部署了兵力，需要最新情报 | 情报+1，了解副官位置 | 被发现，遭遇战 |
| **宗教** | 点燃信标之火 | 要塞塔楼上的古老信标可以驱散不死者，需要重新点燃 | 神选者支持，净化区域 | 亵渎，神选者不满 |
| **补给** | 搜刮军械库 | 要塞军械库中可能还有黑火药和重武器 | 手雷+2，重武器 | 陷阱，爆炸伤害 |

### 特殊任务

| 任务 | 描述 | 奖励 | 惩罚 |
|------|------|------|------|
| **A. 斩首黑栎骑士队长** | 一名强大的黑栎骑士指挥着要塞内的不死者，击杀他将导致敌军混乱 | 士气+2，敌军下地点威胁度-1 | 伤亡惨重，副官介入 |
| **B. 解救被困平民** | 一群平民躲藏在要塞地窖中，需要护送他们撤离 | 新兵+2，士气+1 | 平民死亡，士气-2 |
| **C. 炸毁桥梁** | 要塞外的石桥是不死者大军追击的必经之路，炸毁它可以争取时间 | 时间压力-2，延缓追击 | 炸桥时被困，人员伤亡 |

### 次要任务奖励/惩罚表

| 骰值 | 结果 |
|------|------|
| 1 | 惩罚A：士气-1，城墙部分坍塌伤及人员 |
| 2 | 惩罚B：食物-1，军械库爆炸损失物资 |
| 3 | 惩罚C：1人负伤（2级伤害） |
| 4 | 奖励A：补给+1 |
| 5 | 奖励B：情报+1，发现暗道 |
| 6 | 奖励C：士气+1，获得要塞地图（指挥官情报问题额外+1） |

## 示例地点3：焦土平原（Scorched Plain）

**地点类型**：开阔平原
**环境特征**：焦黑土地、残骸、无遮蔽
**撤退方向**：东

### 标准任务

| 任务类型 | 任务名称 | 描述 | 典型奖励 | 典型惩罚 |
|---------|---------|------|---------|---------|
| **突袭** | 伏击敌军补给线 | 不死者的补给车队穿越平原，设伏袭击 | 士气+1，食物+2 | 被反伏击，伤亡 |
| **侦察** | 确认大军动向 | 西方地平线有不死者的烟尘，确认其数量和方向 | 情报+2，了解追击压力 | 被巡逻队发现 |
| **宗教** | 为阵亡者祈祷 | 平原上遍布上次战役的尸骨，书记官主持仪式安抚亡魂 | 士气+1，神选者支持 | 亵渎，亡灵骚动 |
| **补给** | 搜刮战场残骸 | 上次战役遗落的装备和物资可能还能用 | 装备修复，黑弹+1 | 物资锈蚀，无收获 |

### 特殊任务

| 任务 | 描述 | 奖励 | 惩罚 |
|------|------|------|------|
| **A. 对抗撕裂者先锋** | 撕裂者的一支先锋部队正在平原上集结，先发制人削弱它们 | 撕裂者压力-2，士气+2 | 惨重伤亡，被包围 |
| **B. 建立临时防御工事** | 利用平原上的残骸建立防线，为军团撤退争取时间 | 下地点遭遇骰+1，士气+1 | 工事被突破，人员伤亡 |
| **C. 猎杀奇美拉** | 一只奇美拉在平原上游荡，击杀它可以获得珍贵的炼金材料 | 炼金试剂+3，士气+1 | 伤亡，奇美拉逃脱 |

## 地点生成速查表

### 环境类型与敌人关联

| 环境 | 常见敌人 | 特殊危险 |
|------|---------|---------|
| 沼泽/湿地 | 腐尸、痰尸、变异怖物 | 瘴气、陷阱、迷路 |
| 森林/丛林 | 暗影女巫、斥候型不死者 | 伏击、树影幻觉 |
| 山脉/峡谷 | 黑栎骑士、滚石陷阱 | 落石、寒冷、缺氧 |
| 城市/废墟 | 大量腐尸、杂碎包 | 坍塌、狭窄空间 |
| 平原/旷野 | 骑兵类敌人、奇美拉 | 无遮蔽、大规模冲锋 |
| 河流/渡口 | 水生变异体、溺水腐尸 | 水流、桥梁破坏 |
| 要塞/城堡 | 精英不死者、副官 | 防御工事被敌方利用 |

### 奖励/惩罚设计原则

**奖励梯度**（次要任务6点奖励）：
- 奖励A（骰4）：基础资源（食物/补给/情报）
- 奖励B（骰5）：中等资源+额外好处
- 奖励C（骰6）：士气+稀有物资/战略优势

**惩罚梯度**（次要任务1-3点惩罚）：
- 惩罚A（骰1）：士气下降+人员损失
- 惩罚B（骰2）：资源损失+状态伤害
- 惩罚C（骰3）：纯资源/状态损失，无士气影响

### 特殊任务设计原则

每个地点的3个特殊任务应：
1. **与地点环境强关联**（沼泽→水下任务，要塞→攻城任务）
2. **提供战役级影响**（削弱破碎者、获得强力装备、改变地图状态）
3. **有独特的道德困境**（救平民vs保守秘密，炸桥vs留给友军）
4. **需要情报解锁**，增加指挥官的情报消费决策压力
