# 完整游戏流程示例

> **用途**：展示一节游戏的完整流程，从战役阶段到任务阶段到结算阶段，供AI-GM理解节奏和节点。
> **假设**：4名玩家，分别扮演指挥官、军士长、军需官、重装兵（专家）。另2名NPC专家：医疗兵、狙击手。

---

## 第一节：阿尔德马克边境

### 【战役阶段】

**时间流逝**：Time +1，Pressure +0（起始）

**军需官行动**（玩家A）：选择"获取资源"
- 投战役骰 → 4（部分成功）
- 结果：获得2单位食物，1单位补给
- 军团状态：食物 12 → 14，补给 0 → 1

**间谍总管行动**（GM扮演）：派遣间谍
- 大师间谍"赤烦风"执行长期任务"调查"
- 投骰：2d → 5, 2 → 取高=5 → 2格
- "调查"进度：2/8格

**书记官行动**（GM扮演）：讲述"军团建立的故事"
- 所有参与玩家获得对应属性+1 XP

**指挥官决策**（玩家B）：
- 持有情报：0
- 查看可用任务：突袭敌方前哨、侦察不死者动向、补给搜刮废墟
- 选择**主要任务**：突袭敌方前哨
- 选择**次要任务**：补给搜刮废墟
- 派遣：重装兵+医疗兵+狙击手（主要）；士兵+新兵（次要）

**军士长部署**（玩家C）：
- 主要任务3人：重装兵（lead）、医疗兵、狙击手
- 次要任务2人：士兵、新兵
- 投遭遇骰（主要任务）：1d6 → 2 → **绝望处境**

---

### 【任务阶段：突袭敌方前哨】

**初始处境**：绝望（敌人警觉、数量优势、地形不利）

#### 场景1：接近前哨

**重装兵玩家**："我要带领小队从侧面接近，利用夜色掩护。"

→ **Step 1**：目标 = 悄无声息接近前哨，不被发现
→ **Step 2**：行动 = 组织（技术属性）
→ **Step 3**：处境 = 绝望（敌人警觉，且有巡逻队）
→ **Step 4**：效果 = 有限（1格）。理由：夜幕有效，但巡逻队密集，规模劣势
→ **Step 5**：奖励骰
  - 狙击手："我掩护侧翼，观察巡逻路线" → 协助 → +1d
  - 重装兵：超越自我 → +1d（消耗2压力）
  - 总骰池：组织2级 + 协助1d + 超越1d = 4d
→ **Step 6**：投骰 → 2, 4, 5, 6 → **完全成功（6）**

**结果**：按有限效果执行 = 成功接近前哨，但被发现的风险仍然存在。GM描述：你们摸到了围墙下，但一名巡逻兵似乎听到了动静，正朝这边走来。

#### 场景2：处理巡逻兵

**狙击手玩家**："我瞄准那个巡逻兵的头，一击毙命。"

→ **Step 1**：目标 = 击杀巡逻兵，不发出警报
→ **Step 2**：行动 = 射击（认知属性）
→ **Step 3**：处境 = 危险（目标静止，但黑暗中）
→ **Step 4**：效果 = 一般（2格）。使用瞄准特种行动 → 提升1级 → **极佳（3格）**
→ **Step 5**：奖励骰
  - 重装兵：协助 → +1d
  - 总骰池：射击1级 + 协助1d = 2d
→ **Step 6**：投骰 → 3, 6 → **完全成功（6）**

**结果**：极佳效果 = 巡逻兵被无声击杀，尸体被阴影吞没，警报未触发。进程表"前哨警戒"填3格。

#### 场景3：突入前哨

**医疗兵玩家**："我跟着重装兵冲进去，准备救治伤员。"

→ 领导集体行动：重装兵领导，医疗兵+狙击手参与
→ 参与者各消耗1压力 → 重装兵+2d
→ 重装兵使用武艺（1级）→ 被视为小型队伍
→ 总骰池：组织2级 + 集体2d = 4d
→ 投骰 → 1, 4, 4, 6 → **完全成功（6）**

**结果**：极佳效果（武艺+集体行动）= 小队如利刃般撕开前哨防线，不死者被分割包围。进程表"前哨占领"填3格。

#### 场景4：副官出现

GM："前哨深处传来沉重的脚步声——是'狼人'比希德的副官，一个4级威胁的兽化人！"

**重装兵玩家**："我来挡住他！你们去破坏补给！"

→ 保护行动：重装兵代替队友承受副官攻击的后果
→ 消耗2压力

**狙击手玩家**："我射击他的眼睛！"

→ 威胁度差距：副官4 - 狙击手1 = 3级差距
→ 效果降低3级：极佳→一般→有限→**无效**
→ 使用瞄准 → 无效→**有限**
→ 投骰 → 5 → 部分成功
→ 结果：有限效果 = 填1格威胁表（副官血量：1/8格）
→ 后果：副官反击，狙击手受到2级伤害

**医疗兵玩家**："我给他包扎！"

→ 使用医疗（1级）→ 狙击手暂时无视伤害惩罚（1场景）

#### 场景5：闪回破局

**重装兵玩家**："等一下——闪回到昨天，我在前哨外围设置了绊索陷阱！"

→ 闪回复杂度：中等 = 2点压力
→ 闪回行动：破坏（设置陷阱）→ 投骰 → 6 → 完全成功
→ 结果：副官追击中触发绊索，摔倒，威胁表+2格

**最终**：副官威胁表填满（8格），被击败。前哨被占领。

## 流程节奏总结

```
一节游戏 ≈ 90-120分钟
├── 战役阶段（20-30分钟）
│   ├── 时间流逝
│   ├── 各职位执行行动（军需官、间谍总管、书记官）
│   ├── 指挥官选任务
│   └── 军士长部署 + 投遭遇骰
│
├── 任务阶段（45-60分钟）
│   ├── 3-5个场景
│   ├── 每个场景：玩家声明 → 六步法判定 → 后果
│   └── 可使用闪回、团队协作、恶魔交易
│
└── 结算阶段（15-20分钟）
    ├── 主要任务结算
    ├── 次要任务投骰
    ├── 伤害/压力处理
    ├── 士气调整
    ├── XP分配
    ├── 晋升判定
    └── 在营地场景
```

---

## AI-GM注意要点

1. **节奏控制**：任务阶段每个场景不要太长，保持紧张感
2. **部分成功是常态**：4-5应该占50%以上，驱动叙事发展
3. **主动提示闪回**：玩家卡壳时，主动说"你可以闪回到..."
4. **恶魔交易每轮都要有**：至少提出1-2个交易选项
5. **描述压力状态**：压力3+的角色，在叙事中暗示其焦虑
6. **进程表可视化**：告诉玩家"威胁表还有3格就满了"
