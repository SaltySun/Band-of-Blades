> **用途**：敌人图鉴、威胁度规则、破碎者设定的完整参考。
> **来源**：`L1/刀锋连队_世界观手册.md` + `L1/刀锋连队_核心机制手册.md` + `L1/刀锋连队_AI-GM手册_v1.3.md`

## 1. 威胁度尺度

| 威胁度 | 对应对象 |
|--------|---------|
| **1** | 普通军团兵、新兵、腐尸、普通邪教徒、野生猛兽 |
| **2** | 优质装备专家、精英不死者（暗影女巫、无心者、黑栎骑士普通成员）、痰尸 |
| **3** | 臭名昭著的怖物（哀嚎者、奇美拉、杂碎包）、余烬卫士 |
| **4** | 副官级敌人（"博士"、"狼人"比希德）、攻城武器 |
| **5+** | **仅限**破碎者、余烬之王、神选者（战斗圣女形态） |

### 威胁度影响规则

| 影响项 | 规则 |
|--------|------|
| **效果等级** | 敌人威胁度每比你高1级 → 你的效果等级降1级 |
| **伤害等级** | 敌人威胁度/规模每比你高1级 → 你受到的伤害提升1级 |
| **腐化点数** | 不死者造成的腐化通常等于其威胁度 |
| **经验奖励** | 每次任务XP = 遭遇的最高敌人威胁度 |

### 副官特殊规则

- 副官拥有**2-3个特殊能力**，不仅仅是高威胁度
- 副官在场时，所有普通不死者的威胁度视为+1
- 副官可以执行**撤退**——当威胁表填满一半时，可选择逃离战场（下一场任务必定遭遇）
- 击杀副官 → 士气+2，破碎者战役进度-1

## 3. 破碎者系统（终极Boss）

战役中从三位破碎者中选择**两位**作为终极敌人。

### 凋零者（The Render）

- **主题**：血肉扭曲者，炼金术恐怖，生物恐怖
- **特殊规则**：现代战争（所有使用武器和护甲的部队威胁度+1）
- **副官**："博士"（精于缝制怖物）
- **特色**：血肉工程、瘟疫散布、变形实验

### 破坏者（The Breaker）

- **主题**：风暴女巫，心理恐怖
- **特殊规则**：操控天气与心灵
- **特色**：幻象、恐惧、精神污染

### 撕裂者（The Rend）

- **主题**：战争巨人，极权与反人性
- **特殊规则**：战争机器、强制征召
- **特色**：碾碎一切、军团对抗军团

### 破碎者核心设定

- 原本是9位神选者中的5位
- 被余烬之王腐化后成为追杀人类的怪物
- 每位破碎者拥有独特的恐怖主题和腐化方式
- 战役中会逐渐展现其能力和恐怖

## 4. 破碎者战役阶段出场（扩展推导）

破碎者不是一开始就全力出手的。它们会在战役中**逐步展现**力量，按阶段推进。

### 出场阶段表

| 战役节数 | 凋零者出场 | 破坏者出场 | 撕裂者出场 |
|---------|-----------|-----------|-----------|
| **第1-3节** | 仅通过副官"博士"活动 | 仅天气异常（暴风雨） | 仅征召报告 |
| **第4-6节** | 派出怖物（哀嚎者/奇美拉） | 幻象袭击、精神污染 | 强制征召平民为不死者 |
| **第7-9节** | 亲自出现在战场边缘 | 直接心灵攻击角色 | 率领大军正面逼近 |
| **第10-11节** | 与军团直接交战（非最终战） | 风暴中心对决 | 军团对军团的大战 |
| **第12节** | **最终防守战** | **最终防守战** | **最终防守战** |

### 各破碎者阶段能力

#### 凋零者阶段能力

| 阶段 | 能力 | 效果 |
|------|------|------|
| **早期** | **血肉瘟疫** | 任务中随机1名角色腐化+1（无法抵抗） |
| **中期** | **怖物工厂** | 每场任务额外出现1只随机怖物 |
| **晚期** | **血肉同化** | 击杀的NPC/角色有50%概率被转化为不死者加入敌军 |
| **最终战** | **终极形态** | 凋零者展现真实形态，威胁度5，所有不死者威胁度+1，持续释放瘟疫 |

#### 破坏者阶段能力

| 阶段 | 能力 | 效果 |
|------|------|------|
| **早期** | **低语** | 营地场景中，随机角色压力+1（听到无法解释的低语） |
| **中期** | **幻象袭击** | 任务中，GM可宣布1个场景为"幻象"——成功/失败结果反转 |
| **晚期** | **心灵风暴** | 任务开始时，全队投抵抗骰（决心），失败者压力+2 |
| **最终战** | **意识吞噬** | 破坏者直接攻击角色精神，被攻击者需投抵抗骰，失败则直接获得1个创伤 |

#### 撕裂者阶段能力

| 阶段 | 能力 | 效果 |
|------|------|------|
| **早期** | **征召令** | 每次战役阶段，军团可能损失1名新兵（被征召为不死者） |
| **中期** | **大军压境** | 遭遇骰-1（即更容易投出绝望处境） |
| **晚期** | **战争机器** | 任务中可能出现攻城武器（威胁度4，范围伤害） |
| **最终战** | **碾碎一切** | 撕裂者威胁度5+，其攻击无视护甲，伤害等级自动+1 |

### 最终防守战（第12节）

- **地点**：天刃堡
- **前提**：两位破碎者同时到场
- **胜利条件**：守住天刃堡直到黎明（填满"黎明进程表"12格）
- **失败条件**：天刃堡陷落（危机表满）或神选者死亡且无法复活
- **特殊规则**：
  - 普通角色无法对破碎者造成伤害（只能填进程表）
  - 只有神选者（最终形态）能对破碎者造成直接伤害
  - 黎明进程表：每个完全成功的行动填1格，关键成功填2格
  - 危机表：每次角色死亡/士气归零/食物耗尽填2格

## 5. 余烬之王

- **身份**：不死者大军的终极统治者，腐化的源头
- **威胁度**：6（超越常规尺度）
- **出场**：仅在最终防守战的最后阶段，当两位破碎者都被削弱后出现
- **特性**：
  - 不直接参与战斗，而是作为"背景威胁"存在
  - 每回合增加危机表1格
  - 神选者必须直面余烬之王才能完成战役
- **击败条件**：神选者使用最终形态的能力 + 填满"黎明进程表"
