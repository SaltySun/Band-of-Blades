## 1. 威胁度尺度

| 威胁度 | 对应对象 |
|--------|---------|
| **1** | 普通军团兵、新兵、腐尸、普通邪教徒、野生猛兽 |
| **2** | 优质装备专家、精英不死者（暗影女巫、无心者、黑栎骑士普通成员）、痰尸 |
| **3** | 臭名昭著的怖物（哀嚎者、奇美拉、杂碎包）、余烬卫士 |
| **4** | 副官级敌人（"博士"、"狼人"比希德）、攻城武器 |
| **5+** | **仅限**破碎者、余烬之王、神选者（战斗圣女形态） |

### 威胁度影响规则

| 影响项 | 规则 |
|--------|------|
| **效果等级** | 敌人威胁度每比你高1级 → 你的效果等级降1级 |
| **伤害等级** | 敌人威胁度/规模每比你高1级 → 你受到的伤害提升1级 |
| **腐化点数** | 不死者造成的腐化通常等于其威胁度 |
| **经验奖励** | 每次任务XP = 遭遇的最高敌人威胁度 |

- 威胁度2怪物 → **8格**

## 3. 破碎者系统（终极Boss）

战役中从三位破碎者中选择**两位**作为终极敌人。

### 凋零者
- **主题**：血肉扭曲者，炼金术恐怖，生物恐怖
- **特殊规则**：现代战争（所有使用武器和护甲的部队威胁度+1）
- **副官**："博士"（精于缝制怖物）
- **特色**：血肉工程、瘟疫散布、变形实验

### 破坏者
- **主题**：风暴女巫，心理恐怖
- **特殊规则**：操控天气与心灵
- **特色**：幻象、恐惧、精神污染

### 撕裂者
- **主题**：战争巨人，极权与反人性
- **特殊规则**：战争机器、强制征召
- **特色**：碾碎一切、军团对抗军团

### 破碎者核心设定
- 原本是9位神选者中的5位
- 被余烬之王腐化后成为追杀人类的怪物
- 每位破碎者拥有独特的恐怖主题和腐化方式
- 战役中会逐渐展现其能力和恐怖
