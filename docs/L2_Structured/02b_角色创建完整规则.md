# 角色创建完整规则（基于PDF角色卡 v1.3 权威版）

> **用途**：与PDF角色卡完全一致的角色创建参考。当本文档与 `01_核心机制.md` 或 `02_角色系统.md` 冲突时，**以本文档为准**。
> **来源**：PDF角色卡（刀锋连队_角色卡 v1.3）直接提取整理。
> **与bos项目的重大差异**：本文档基于PDF权威数据，修正了bos项目（Blades of the Shattered）中的多项错误。

---

## 一、创建流程总览（4步）

所有军团士兵（专家/士兵/新兵）的创建流程相同：

```
Step 1: 选择一个起始能力（不知道选什么就选第一个）
Step 2: 选择文化 → 选一个名字，两个特性，并描述它们
Step 3: 分配4点行动点数（起始最大等级为2）
Step 4: 报道并服役
```

> **与bos项目的差异**：bos项目使用6点自由分配和3级起始上限，PDF权威为**4点**和**2级上限**。

## 三、Step 2: 文化背景（4选1，选2特性）

### 3.1 四种文化

| 文化 | 描述 | 可选特性（4选2） |

### 3.2 特性对行动上限的影响

| 特性 | 影响 |
|------|------|
| 和蔼可亲 | 社交上限 = 2 |
| 学识渊博 | 调查上限 = 2 |

## 四、Step 3: 行动等级分配

### 4.1 分配规则（PDF权威）

- **总点数**：**4点**
- **分配方式**：自由分配到11种行动中
- **初始状态**：所有行动从0开始
- **起始最大等级**：**2**（不是3！）
- **上限**：默认3，受文化特性影响

> **与bos项目的重大差异**：bos项目使用6点自由分配，PDF权威为**4点**。起始最大等级bos写3，PDF写**2**。

### 4.2 11种行动

| 行动（中文） | 行动（英文key） | 属性 | 描述 |
|-------------|----------------|------|------|
| 调查 | investigation | 认知 | 使用细致观察收集信息，调查人、文件或物品 |
| 射击 | marksmanship | 认知 | 远处朝目标射击，寻找清晰射击线 |
| 修补 | rigging | 认知 | 机械化解决方案，修理机械，解除陷阱，撬锁 |
| 社交 | sway | 认知 | 获得资源、信息，或留下好印象 |
| 侦查 | scout | 认知 | 侦察地点收集信息，不被察觉地移动或观察 |
| 训诫 | discipline | 认知 | 训诫士兵，利用人格力量强迫服从 |
| 游击 | skirmish | 技术 | 近身搏斗，突袭，摔跤，近距离开枪 |
| 破坏 | wreck | 技术 | 爆破破坏，散播混乱，使用爆炸物 |
| 机动 | maneuver | 技术 | 移动、攀爬、游泳、奔跑、驾驭动物 |
| 组织 | command | 技术 | 组织小队协同行动，对目标协同开火 |
| 动摇 | resolve | 决心 | 用欺骗、魅力、逻辑动摇某人；伪装 |

## 五、Step 4: 装备系统

### 5.1 负载等级（3选1）

| 负载 | 特点 | 说明 |
|------|------|------|
| 轻装 | 轻便且快捷 | 包括基础物品 |
| 中装 | 轻装所有物品 + 额外物品 | 均衡配置 |
| 重装 | 所有中装物品 + 额外物品 | 缓慢 |

### 5.2 功能栏

- **每位角色有2个功能栏**
- 从角色预设装备列表中选择物品填入功能栏
- 某些物品占用2个功能栏（如黑弹、圣骸、重甲）
- 加粗字体为优质物品

### 5.3 护甲

| 护甲类型 | 效果 | 占用功能栏 |
|---------|------|-----------|
| 护甲（皮甲/链甲混搭） | 抵抗物理攻击时1护甲 | 不占 |
| 盾牌 | 抵抗物理攻击时1护甲 | 不占 |
| 重甲（金属板甲） | 抵抗物理攻击时2护甲 | 占2格，无优质版 |
| 优质护甲 | 同上，更精良 | — |
| 优质盾牌 | 同上，金属铸造 | — |

### 5.4 通用装备清单

**标准物品**：护甲、单手武器、重型武器、盾牌、冬装、兵粮

**套装**：士兵套装、破坏套装、修理套装、攀爬套装、烹饪套装、枪支保养套装、医疗套装

**特殊物品**：
- 圣骸（占用2格，3次使用）：降低腐化1点，抵抗腐化+1d。优质版占用3格，降低2点
- 黑弹（占用2格，3次使用）：对抗不死者时行动获得效力
- 黑箭（3次使用）：箭头涂黑弹粉尘，攻击不死者获得效力
- 油（3次使用）：润滑、填充油灯、易燃
- 信号枪（4发）：带色烟幕弹和闪光信号
- 透镜组：数片水晶透镜，看远处
- 罗盘与地图：本地地图加个人注解
- 书本与卷轴（2次使用）：为调查+1d或+1效果等级
- 家乡纪念物、战旗、奢侈物品等

**各角色专属装备**：详见各角色卡预设列表

## 六、XP与晋升系统

### 6.1 XP获取规则

每次任务结束时，对于以下每一项标记1XP：

| 条件 | XP |
|------|-----|
| 在任务中存活 | +1 |
| 帮助你的团队（因职业而异，见各角色XP条件） | +1 |
| 表现你的文化或创伤 | +1（两者都有则+2） |
| 敌人最高威胁每有一级 | +1 |

**额外XP**：当你在绝望处境下掷行动骰时，在对应属性下标记1XP。

### 6.2 晋升路径

```
新兵 ──填满XP表──→ 士兵 ──填满XP表──→ 专家
  │                    │                    │
  │                    │                    └── 继续选新能力
  │                    │
  │                    └── 可选"专家训练"晋升专家
  │                        （选新手册+1级特种行动+1能力）
  │
  └── 获得1级忍耐+1项士兵能力（新兵不再）
```

**各阶段晋升详情**：

| 阶段 | 触发条件 | 获得 |
|------|---------|------|
| 新兵→士兵 | 第一次填满XP表 | 1级忍耐 + 1项士兵能力（必须选"新兵不再"作为第一次提升） |
| 士兵→专家 | 填满XP表后选择"专家训练" | 选择新游戏手册 + 对应专家行动提升1级 + 1项专家能力 |
| 专家继续 | 填满XP表 | 选1项新能力（可从当前手册或其他手册中选择） |

> **与bos项目的差异**：bos项目说新兵3次任务晋升，PDF权威为**填满XP表**晋升。士兵也有"专家训练"可选能力来晋升专家。

## 七、职位系统（战役管理层）

**重要**：职位（指挥官、军需官、军士长、书记官、间谍总管）是**战役管理层**，与特种士兵角色**独立**。PDF角色卡中有独立的职位卡。

| 职位 | 类型 | 说明 |
|------|------|------|
| 指挥官 | 必选 | 军团最高决策者。选择任务路线、决定军团是否进军 |
| 军士长 | 必选 | 军团纪律与士气维护者。分配遭遇骰、决定任务人选 |
| 军需官 | 必选 | 军团物资与补给管理者。执行战役行动（放假、募兵、补给等） |
| 书记官 | 可选 | 军团故事记录者。记录编年史、讲述故事获得增益 |
| 间谍总管 | 可选（3人以上） | 管理间谍网络，派遣短期/长期任务 |

**职位特性**：每个职位在游戏前从文化、风评、个性、外形中选择特性。

> **与bos项目的重大差异**：bos项目在角色创建时绑定职位（5选1，去重）。PDF权威中职位是独立的战役管理角色，不由特种士兵创建时选择。

## 九、与文档库v1.3及bos项目的差异汇总

| 项目 | PDF角色卡权威 | bos项目 | 文档库v1.3 |

| 文化背景 | ✅ 有（4文化×4特性，选2） | ✅ 有（4文化×4特性，选2） | ❌ 无 |
