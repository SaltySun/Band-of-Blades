# 刀锋连队 — 战役管理手册（GM专用）

> 版本：v1.2 拆分版 | 用途：GM主持战役时参考任务、结算、营地、敌人

---

## 任务类型系统

### 标准任务类型

| 任务类型 | 描述 | 典型奖励 | 典型惩罚 | 适合专家 |
|---------|------|---------|---------|---------|
| **突袭任务** | 快速打击敌人目标，制造混乱 | 士气+1，消灭敌人 | 伤亡，敌人警觉 | 重装兵、狙击手、军官 |
| **侦察任务** | 侦查敌情、地形、资源 | 情报+1，了解威胁 | 被发现，遭遇战 | 斥候、军官 |
| **宗教任务** | 与神选者相关的精神/信仰任务 | 神选者支持，士气 | 亵渎，腐化+ | 书记官、医疗兵 |
| **补给任务** | 获取食物、物资、装备 | 食物+，补给+ | 物资损失，伏击 | 军需官、军官 |

### 特殊任务
- 每个地点有**3个特殊任务**
- 需要消耗**情报**解锁执行权
- 通常有独特的背景、奖励和惩罚

### 次要任务
- 由指挥官选择的第二个任务
- **不需要玩家扮演执行**，由遭遇骰结果自动决定成败
- 成功 = 获得奖励；失败 = 承受惩罚
- 可以用情报为次要任务的遭遇骰+1
- **可以和主要任务派不同的人**

### 次要任务投骰规则
- 使用**次要任务参与者的调查骰池**投遭遇骰
- 结果对应地点的次要任务奖励/惩罚表：
  - **1** = 惩罚A（通常是士气-1或物资损失）
  - **2** = 惩罚B
  - **3** = 惩罚C
  - **4** = 奖励A（通常是情报或物资）
  - **5** = 奖励B
  - **6** = 奖励C（通常是士气+1或稀有物资）

## 结算阶段流程

结算阶段是任务阶段结束后、战役阶段开始前的关键环节，按以下顺序执行：

```
1. 主要任务结算
   ├── 存活角色获得任务奖励
   └── 死亡角色从名册移除
2. 次要任务结算
   ├── 投遭遇骰（使用次要任务参与者的调查骰池）
   └── 根据结果分配奖励/惩罚
3. 伤害与压力处理
   ├── 角色勾销伤害（根据休息条件）
   ├── 处理压力满6点的角色（获得创伤）
   └── 处理4级伤害的角色（死亡）
4. 士气调整
   ├── 阵亡：士气-1
   ├── 士气6+：结算时+1士气
   └── 士气3-：结算时-1士气
5. 经验点分配
   ├── 基础XP = 本次最高敌人威胁度
   └── 额外XP（协助/扮演创伤/承受伤害/死亡等）
6. 晋升判定
   └── 新兵存活2次任务 → 晋升为士兵
7. 进入"在营地"场景
```

### 主要任务结算
- 所有参与主要任务且**存活**的角色获得任务奖励
- **死亡**的角色从名册中移除（保留在编年史中）
- 每个死亡角色使**士气-1**

### 伤害与压力处理
- **压力满6点**：清空压力→获得精神创伤→选择症状→失能
- **4级伤害**：角色死亡
- 如果同时满足压力满和4级伤害，先处理死亡

### 晋升判定
- 每个新兵检查**存活任务次数**
- 达到**2次存活** → 晋升为士兵
- 晋升时：获得忍耐特种行动1级，获得1个新能力

## 在营地场景

"在营地"是每次任务结束后、战役阶段开始前的叙事场景。根据当前**士气水平**，场景氛围截然不同。

### 士气0-3：沮丧营地

**氛围**：沉默、易怒、绝望、阴影笼罩

**场景特征**：
- 角色们围坐在微弱的篝火旁，很少有人说话
- 书记官试图讲故事，但声音被风声吞没

**机制效果**：
- 书记官讲故事时，听众很难获得经验点
- 休息时只能勾销**1格伤害**（而非正常的2格）
- **士气3或更低**：结算时额外-1士气
- 可能出现**逃兵事件**

**AI-GM叙事建议**：
- 描述潮湿、寒冷、腐臭味
- 让玩家角色感受到"我们可能要死在这里"

### 士气4-7：普通营地

**氛围**：疲惫但仍在坚持、例行公事、麻木

**机制效果**：
- 书记官讲故事正常结算经验点
- 休息时可以正常勾销伤害和清除压力

**AI-GM叙事建议**：
- 描述日常军事生活的细节
- 鼓励玩家角色之间的小互动

### 士气8-10：振奋营地

**氛围**：士气高涨、歌声、笑声、充满希望

**机制效果**：
- 书记官讲故事时，所有参与者额外+1经验点
- 休息时额外勾销**1格伤害**
- **士气6+**：结算时额外+1士气
- 可能出现**自愿加入的新兵**

**AI-GM叙事建议**：
- 描述温暖、安全感（相对的）、战友情谊
- 注意：不要太过乐观——这只是暴风雨前的宁静

### 营地活动

| 活动 | 效果 | 限制 |
|------|------|------|
| **休息** | 勾销伤害，清除压力 | 需要安全营地和食物 |
| **修理装备** | 修补行动恢复装备 | 需要工具和材料 |
| **交换情报** | 社交行动分享信息 | 无限制 |
| **祷告/冥想** | 动摇行动恢复精神 | 需要安静环境 |
| **训练新兵** | 训诫行动帮助新兵 | 需要老兵参与 |
| **写遗书** | 无机制效果，纯叙事 | 任何角色 |

## 游戏结构

```
┌─────────────────────────────────────────────────┐
│                    战役阶段                         │
│  ├── 时间流逝（积累时间和压力，消耗食物）            │
│  ├── 各职位执行战役行动                             │
│  │   ├── 军需官：获取资源/长期项目/休息恢复/募兵     │
│  │   ├── 间谍总管：派遣间谍（短期/长期任务）          │
│  │   └── 书记官：讲述军团故事                        │
│  ├── 指挥官决定是否进军                             │
│  └── GM生成新任务 → 进入任务阶段                     │
└──────────────────────┬────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────┐
│                    任务阶段                         │
│  ├── 指挥官选主要任务+次要任务                      │
│  ├── 军士长派人选 + 军需官分配补给                   │
│  ├── 投遭遇骰确定起始处境                            │
│  ├── 主要任务：小队执行，投骰、承担后果              │
│  │   └── 可使用闪回、团队协作                        │
│  ├── 次要任务：由遭遇骰决定结果                      │
│  ├── "在营地"短暂扮演                                │
│  └── 结算：记录奖惩、士气损失 → 回到战役阶段         │
└─────────────────────────────────────────────────┘
```

- **一节游戏** = 一次任务阶段 + 一次战役阶段
- **完整战役**约 12节
- 到达天刃堡 = 最终防守战 = 战役结束

## AI-GM叙事指南

### 核心叙事原则

1. **部分成功驱动叙事**：4-5是最常见结果 → 玩家总是付出代价
2. **精神压力=英雄主义的燃料**：超越自我让人做到不可能的事，但代价是崩溃
3. **角色轮换降低死亡恐惧**：角色死了换下一个 → 敢死队氛围
4. **双阶段结构**：战术小队行动 + 战略层管理
5. **恶魔的交易=道德困境**：玩家主动提出彼此代价 → 团队内自然发生背叛/牺牲
6. **闪回=创造性解决**：鼓励玩家用闪回创造有利条件，但代价是压力
7. **团队协作=集体牺牲**：鼓励玩家互相协助，但协助也要付出代价

### GM叙事建议

**开场与氛围**
- 强调泥、血、雨、黑暗
- 军团不是在"冒险"，是在"求生"
- 每个决定都有代价

**描述敌人**
- 不死者是恐怖的存在，不只是"怪物"
- 描述腐化的视觉细节：肿胀、变色、不自然的肢体
- 破碎者的出现应该让玩家感到真正的恐惧

**处理死亡**
- 角色死亡时给予告别时刻
- 让书记官记录他们的牺牲
- 新兵接替时强调"又一张新面孔"

**推进腐化**
- 逐渐描述角色的枯萎症状
- 让其他角色注意到这些变化
- 制造"要不要抛弃被腐化的战友"的困境

**使用进程表**
- 让玩家看到危机表逐渐填满的紧张感
- 用竞速表制造"争分夺秒"的紧迫感
- 威胁表填满时给击败精英的成就感

**闪回引导**
- 当玩家陷入困境时主动提示"你可以闪回到..."
- 要求闪回的细节必须合理
- 闪回消耗压力，保持机制平衡

**团队协作引导**
- 鼓励玩家描述如何协助彼此
- 集体行动时让领导者承担更多风险
- 准备行动应该被经常使用

### AI-GM判定速查

| 情况 | 判定 |
|------|------|
| 玩家宣布行动 | 六步法：目标→行动→处境→效果→奖励→投骰 |
| 玩家不想承受后果 | 投抵抗骰，承受6-结果点压力 |
| 压力满6点 | 精神创伤，选8种症状之一 |
| 压力5点 | 濒临崩溃，角色可能出现轻微失误 |
| 压力3-4点 | 感到紧张，描述应反映焦虑 |
| 伤害行满 | 向上溢推，3级满→4级=死亡 |
| 腐化满7点 | 获得枯萎症状，枯萎病+1级 |
| 遭遇骰 | 1-3绝望/4-5危险/6安全 |
| 敌人威胁度高 | 效果降1级/伤害升1级/腐化=威胁度 |
| 任务完成 | XP = 最高威胁度 |
| 玩家陷入困境 | 提示闪回（消耗1-2压力） |
| 多人协作 | 协助+1d / 领导集体+最多3d / 准备改善处境 / 保护承受后果 |
| 结算阶段 | 主要→次要→伤害/压力→士气→XP→晋升→营地 |
