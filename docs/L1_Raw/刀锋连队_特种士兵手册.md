## 重装兵

### 定位
保护者 / 近战斗士

### 特种行动：武艺
- 自带**1级武艺**
- 使用时无需投骰，次数 = 等级
- 效果：在战斗中被视为**一支小型队伍**（规模优势无效）
- 可消耗次数作为小队特殊护甲

### 能力列表（选多项）

| 能力 | 效果 |
|------|------|
| **堡垒** | 消耗武艺次数作为小队抵抗后果用的特殊护甲 |
| **援助** | 保护队友时抵抗骰+1骰；协助队友超越自我只+1压力 |
| **顽强** | 伤害带来的惩罚降低1级（4级及以上除外） |
| **武器大师** | 近身战斗超越自我时获得效力 |
| **战争机器** | 超越自我时可降低敌人威胁度1级，或进行超人行动 |
| **精力充沛** | 休息时额外勾销1格伤害；受伤时减少1点精神压力 |
| **对抗黑暗** | 所有能看到你的队友抵抗恐惧和腐化时+1骰 |
| **精英** | 精通两种行动（提升上限提升至4级） |

## 医疗兵

### 定位
战地医生 / 急救专家

### 特种行动：医疗
- 自带**1级医疗**
- 使用时无需投骰，次数 = 等级
- 效果：令一个军团成员在**一个场景时间内**暂时无视伤害惩罚
- 4级伤害依然是致命的

### 能力列表

| 能力 | 效果 |
|------|------|
| **特派员** | 可参与专家名额已满的任务 |
| **急救** | 使用1次补药移除队友1级伤害 |
| **时辰未到** | 消耗1次医疗将4级伤害降为3级，避免死亡（必须尽快使用） |
| **绝命毒医** | 使用1次补药令队友的某次肉体行动获得效力 |
| **战地止血** | 任务结束时消耗剩余医疗次数勾销队友2-3级伤害（每人限一次） |
| **化学家** | 可选择炼金武装带作为标准物品（奥尔医用炼金试剂） |
| **精神支持** | 每次任务一次，讲故事让听众降低1点精神压力 |
| **精英** | 精通两种行动（提升上限提升至4级） |

### 奥尔医用炼金试剂（化学家专属）

| 试剂 | 效果 |
|------|------|
| **枭目油** | 24小时夜视（光敏） |
| **化合软膏** | 中和绝大多数毒素 |
| **深海** | 一个场景时间内不用呼吸 |
| **暴怒毒素** | 肉体行动获得效力，但需决心抵抗避免敌我不分（有成瘾性） |

## 军官

### 定位
领导者 / 战术家

### 特种行动：渠道
- 自带**1级渠道**
- 效果：通过闪回，利用关系获取额外资源
- **资源质量由动摇骰决定**：
  - **1-3**：有限质量（劣质物资）
  - **4-5**：一般质量（标准物资）
  - **6**：优质质量（特殊物品）
- 可消耗额外渠道次数提升资源质量

### 能力列表

| 能力 | 效果 |
|------|------|
| **团结** | 命令一名队友无视伤害惩罚行动一次 |
| **纪律** | 训诫行动等级上限为4 |
| **后勤** | 渠道等级上限为4 |
| **战术撤退** | 任务中一次，命令全队撤退并重新部署 |
| **激励演讲** | 任务开始时全队+1遭遇骰 |
| **权威** | 社交和训诫行动获得效力 |
| **老兵经验** | 绝望处境行动+1骰 |
| **精英** | 精通两种行动（提升上限提升至4级） |

## 狙击手

### 定位
远程杀手 / 精准射手

### 特种行动：瞄准
- 自带**1级瞄准**
- 效果：每次使用可将射击行动效果等级**提升1级**
- 对抗高威胁度怪物时极为有效
- 可将效果0（不可能）变为可能

### 能力列表

| 能力 | 效果 |
|------|------|
| **鹰眼** | 瞄准等级上限为4 |
| **致命精准** | 射击效果**额外+1级**（使用瞄准时） |
| **隐蔽射击** | 射击不会暴露位置（一次） |
| **冷静** | 压力满时仍可选择承受1创伤行动 |
| **标记目标** | 标记一个敌人，队友对其攻击+1骰 |
| **快速装填** | 射击失败时可用压力重投 |
| **孤立无援** | 单独行动时效果极佳 |
| **精英** | 精通两种行动（提升上限提升至4级） |

## 士兵

### 定位
饱经风霜的老兵

### 特种行动：忍耐
- 自带**1级忍耐**（士兵专属）
- 效果：**一次抵抗骰+2骰**
- 必须在投骰之前宣布抵抗
- 每次投骰只能使用1次忍耐

> **完整描述**：当你使用忍耐时，你会通过你在军旅生涯中所学到的所有来之不易的经验忍受战争所能带来的一切痛苦。忍耐是士兵专属的特种行动。使用忍耐可以允许你的**一次抵抗骰+2骰**。你必须在投骰之前宣布抵抗，但带来的加骰有几率改变生死。每次投骰只能使用1次忍耐。

### 其他能力
- **老兵**：伤害惩罚时仍投完整骰池（非劣势）
- **战友**：协助队友时+1骰
- **不屈**：压力5-6时仍可正常行动
- **适应力**：可临时使用一种未受训行动
- **警戒**：任务开始时全队感知投骰+1骰
- **牺牲**：为队友承受一次伤害
- **精英**：精通两种行动（提升上限提升至4级）

### 压力/创伤上限提升
- **"幸存者"**（可选能力）：压力上限从6提升至8/10
- **"老兵"**（可选能力）：创伤承受上限从2提升至3/4

## 新兵

### 定位
未经考验的新人

### 特性
- 威胁度1
- 无特种行动

### 能力
- **潜力**：首次晋升为士兵时额外获得1能力
- **求生欲**：压力首次满时可选择清空压力而非受创伤
- **学习**：使用被协助的行动时+1骰
- **运气**：任务中一次重投任意骰
- **热情**：每次任务首次行动+1骰
- **成长**：存活**2次任务**后晋升为士兵

### 晋升
- 新兵存活2次任务 → 晋升为士兵
- 晋升时获得1个新能力
- 晋升后获得忍耐特种行动
