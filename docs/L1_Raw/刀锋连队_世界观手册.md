## 神选者系统

### 三选一

#### 施芮亚（巴尔塔治愈女神）
- 主题：军事与战略
- 初始威胁度：4
- 初始情报：每完成2次主要任务获得1份情报（来自有角者外出探索观察）

#### 有角者（潘雅丛林之神）
- 主题：神秘力量与变形
- 初始情报：1份（带回敌人部队配置和地形信息）

#### 左拉（泽姆亚远古神明）
- 主题：强力正面作战
- 初始情报：0（军团没时间侦察）

### 神眷形态
神选者拥有多种形态/能力，按战役进度解锁：
- **施芮亚**：阿斯莉卡之慈悲/佑/泪 → 受膏者 → 战斗圣女（**神选者之血**，血液成为圣物） → 战争圣徒
- **有角者**：赏/眼/筋 → 变形者 → 受膏者 → 伟大猎手/森林之翼/白腿马之皮
- **左拉**：晨星 → 圣印 → 永生神之怒/吻/力

### 神选者战役规则
- 神选者不能简单死去（可牺牲神眷等级避免死亡）
- 每次任务只能使用一次神选者能力
- 神选者不受腐化影响
- 神选者可以执行**超凡使命**（单人高难度任务）

### 超凡使命
- 神选者可以执行**单人高难度任务**
- 不需要其他角色参与
- 通常涉及对抗破碎者的直接行动
- 失败后果严重（神选者受伤/被俘/神眷等级下降）
- 成功可大幅削弱破碎者或扭转战局

## 破碎者系统

### 三选二（终极Boss）

#### 凋零者
- **主题**：血肉扭曲者，炼金术恐怖，生物恐怖
- **特殊规则**：现代战争（所有使用武器和护甲的部队威胁度+1）
- **副官**："博士"（精于缝制怖物）
- **特色**：血肉工程、瘟疫散布、变形实验

#### 破坏者
- **主题**：风暴女巫，心理恐怖
- **特殊规则**：操控天气与心灵
- **特色**：幻象、恐惧、精神污染

#### 撕裂者
- **主题**：战争巨人，极权与反人性
- **特殊规则**：战争机器、强制征召
- **特色**：碾碎一切、军团对抗军团

### 破碎者核心设定
- 原本是9位神选者中的5位
- 被余烬之王腐化后成为追杀人类的怪物
- 每位破碎者拥有独特的恐怖主题和腐化方式
- 战役中会逐渐展现其能力和恐怖

## 敌人图鉴

### 威胁度尺度

| 威胁度 | 对应对象 |
|--------|---------|
| **1** | 普通军团兵、新兵、腐尸 |
| **2** | 精英不死者（暗影女巫、无心者、黑栎骑士） |
| **3** | 臭名昭著的怖物（哀嚎者、奇美拉、杂碎包） |
| **4** | 副官（"博士"、"狼人"比希德） |
| **5+** | **仅限**破碎者、余烬之王、神选者 |
