## 一、战役总览与核心调性

### 一句话设定
刀锋军团是一支溃败的雇佣兵团，在西方不死者大军的追击下向东撤退，目标是穿越阿尔德马克到达天刃堡进行最后的决战。

### 核心调性
- **黑暗军事奇幻**：文艺复兴晚期科技水平（火绳枪、黑火药、炼金术）
- **没有救世主**：没有精灵矮人，没有救世主魔法
- **只有鲜血、泥坑、漫长的行军，和必死的觉悟**

### 主题
这款游戏的核心不是"打赢战争"，而是**"在注定失败的撤退中，决定你愿意为什么牺牲，以及你能守护住多少人性"**。机制上每一个设计——精神压力、恶魔的交易、腐化、角色轮换——都在服务于这个主题。

### 关键数字
- 完整战役约 **12节**（12次任务+战役循环）
- 到达天刃堡 = 最终防守战 = 战役结束
- 科技水平：**文艺复兴晚期**

## 二、核心骰子机制

### 基本投骰
- 投若干 **d6**，取**最大值**作为结果
- **6** = 完全成功
- **4-5** = 部分成功（做到了，但有代价/后果）
- **1-3** = 失败（状况变糟）
- **两个以上6** = 关键成功（额外优势）
- 骰池为0或负数时，投**两个骰子取较小值**（无法关键成功）

### 五种骰子类型
| 骰子类型 | 用途 | 投骰时机 |
|---------|------|---------|
| **行动骰** | PC进行有挑战性的行动 | 玩家声明行动时 |
| **抵抗骰** | 减轻或避免后果 | 玩家不想接受后果时 |
| **遭遇骰** | 任务开始时确定处境 | 任务部署完成后，GM投 |
| **战役骰** | 战役阶段的职位行动 | 战役阶段各职位执行行动时 |
| **幸运骰** | GM判断不确定事件 | GM需要随机决定时 |

### 11种通用行动
| 属性 | 行动 | 适用场景 |
|------|------|---------|
| **认知** | 调查、射击、修补、社交、侦查、训诫 | 信息收集、远程战斗、设备维护、谈判、侦察、纪律 |
| **技术** | 游击、破坏、机动、组织 | 近战骚扰、摧毁设施、移动/驾驶、指挥调度 |
| **决心** | 动摇 | 抵抗恐惧、坚守阵地、鼓舞士气 |

### 奖励骰来源（最多+2d）
1. **协助**（其他玩家角色协助）：+1d
2. **超越自我**：+1d 或 +1效果等级（消耗2点压力）
3. **恶魔的交易**：其他玩家或GM提出代价，换取+1d

## 三、闪回机制

> **补充说明**：闪回是任务阶段中的核心叙事工具，允许玩家临时插入过去场景来解决当前困境。手册v1.0遗漏此系统，现补充。

### 什么是闪回
闪回允许玩家在任务进行中**临时插入一个之前发生过的场景**。玩家可以闪回到任务开始前或任务中的某个时刻，执行一个行动来解决当前困境。

### 闪回消耗
- 闪回消耗 **1-2点精神压力**（由GM根据复杂度判断）
- 简单闪回（"我之前藏了一把刀"）= 1点压力
- 复杂闪回（"我早上侦查了巡逻路线并做了标记"）= 2点压力

### 闪回限制
- **不能**闪回到过去改变已经确定的事实
- **不能**闪回到"不可能"的时间点（如"我提前杀死了这个敌人"）
- 闪回中的行动**仍需投骰**
- 闪回不能用来直接跳过障碍，只能用来创造解决障碍的条件

### 闪回常见用途
| 用途 | 范例 | 压力消耗 |
|------|------|---------|
| **提前侦查** | "闪回到今天早上，我侦查了巡逻路线" | 2点 |
| **设置陷阱** | "闪回到出发前，我在门后设置了绊索" | 1-2点 |
| **准备物品** | "闪回到昨天，我偷了看守的钥匙" | 1点 |
| **建立关系** | "闪回到上周，我贿赂了城门守卫" | 2点 |
| **获取情报** | "闪回到进军前，我打听了当地邪教徒的弱点" | 2点 |

### AI-GM闪回指引
- **主动提示**：当玩家陷入困境且无法直接解决时，提示"你可以闪回到..."
- **要求细节**：让玩家描述闪回场景的具体内容
- **判断合理性**：闪回必须合理，不能改变已确定的事实
- **消耗压力**：确保闪回消耗压力，维持机制平衡
- **闪回后果**：闪回中的失败也会带来后果（如被提前发现）

## 四、行动骰六步法——GM判定流程

当玩家宣布要进行有挑战性的行动时，按以下步骤执行：

### Step 1：玩家宣布目标
玩家清晰说出**要做什么**和**期望的结果**。如果目标不清晰，要求玩家澄清。

### Step 2：玩家选择行动类型
玩家从11种通用行动中选择最合适的一项。

> **AI-GM判断**：如果玩家的描述不符合任何通用行动，允许使用最接近的一项，或要求玩家调整描述。

### Step 3：GM设定处境
根据局势判断初始处境：

| 处境 | 判断标准 | 失败后果 |
|------|---------|---------|
| **安全** | 玩家有压倒性优势、敌人毫无防备、玩家完全掌控局面 | 小麻烦/效果降低 |
| **危险（默认）** | 势均力敌、双方都有机会、标准对抗 | 伤害/麻烦/处境恶化 |
| **绝望** | 玩家处于明显劣势、敌人强大且警觉、时间紧迫 | 重伤/严重麻烦/死亡风险 |

> **AI-GM提示**：大部分情况用"危险"。只有明显优势才用"安全"，明显劣势才用"绝望"。在绝望处境中行动，对应属性+1经验点。

### Step 4：GM设定效果等级
基于**效果三要素**判断：

- **效力**：针对弱点、额外时间、更大风险（二元：有效/无效）
- **规模**：敌人数量、区域大小（规模优势可恶化处境、降低效果、提升伤害）
- **威胁**：装备质量/生物危险度（普通=1，优质=2，重武器=3+）

| 效果等级 | 格子数 | 含义 |
|---------|--------|------|
| **极限** | 4+ | 超乎想象的好 |
| **极佳** | 3 | 额外的好成果 |
| **一般** | 2 | 正常预期成果 |
| **有限** | 1 | 部分或较弱结果 |
| **无效** | 0 | 毫无效果 |

> **AI-GM提示**：先问"能做到吗？"（有无效力），再问"能做到多好？"（效果等级）。威胁度差距每差1级，效果降低1级。

### Step 5：玩家添加奖励骰
玩家可以选择：
- **协助**：另一名角色协助，+1d
- **超越自我**：+1d 或 +1效果等级（消耗2点压力）
- **恶魔的交易**：接受GM或其他玩家提出的代价，+1d（最多+2d）

> **AI-GM提示**：主动提出恶魔的交易。代价可以是：连带损伤、背叛朋友、激怒神选者、承受腐化、推进危机进程表等。交易不论成败都会生效。

### Step 6：投骰与判断
根据最终骰池投骰，判断结果：
- **6** = 完全成功，按效果等级执行
- **4-5** = 部分成功，按效果等级执行但附加1个后果
- **1-3** = 失败，状况变糟，附加1-2个后果
- **两个6+** = 关键成功，额外获得优势

## 五、团队协作系统

> **补充说明**：手册v1.0只提到"协助"，但PDF中有完整的4种团队协作类型。现补充完整系统。

当多名角色参与同一次行动或互相支援时，使用以下协作规则：

### 1. 协助（Assist）
- **描述**：你如何帮助另一位PC
- **消耗**：**1点精神压力**
- **效果**：给被协助者 **+1骰**
- **限制**：每次行动骰只能接受1位角色的协助
- **风险**：协助者可能承受行动后果（GM判断）

> **范例**：重装兵举起盾牌掩护狙击手射击 → 狙击手射击+1d，重装兵承受敌人反击风险

### 2. 领导集体行动（Lead Group Action）
- **场景**：多位PC做同一件事（全队冲锋、集体侦察、共同防守）
- **机制**：一人**领导**，投骰；其他参与者各消耗**1点压力**增加**+1骰**（最多+3骰）
- **风险**：如果领导者失败，**所有参与者承受后果**
- **优势**：领导者可叠加自身能力加成

> **范例**：军官领导全队冲锋 → 军官投组织骰，3名队员各消耗1点压力+1d，共+3d；如果失败全队都受伤

### 3. 准备行动（Setup Action）
- **场景**：为队友创造有利条件，不直接解决障碍
- **效果**：接下来所有配合你的角色**改善1级处境**或**提升1级效果**
- **消耗**：通常无需额外消耗，但可能承受后果

> **范例**：
> - 狙击手"我来压制敌人的火力" → 射击投骰 → 成功 → 接下来队友机动的处境从危险改善为安全
> - 斥候"我先侦查了安全路线" → 调查投骰 → 成功 → 接下来队友沿此路线行动效果+1级

### 4. 保护（Protect）
- **场景**：当队友承受后果时，挺身而出代替承受
- **消耗**：**1-2点压力**（取决于后果严重程度）
- **效果**：原目标不受后果，保护者承受全部后果

> **范例**：暗影女巫的精神攻击即将击中医疗兵 → 重装兵"我来挡" → 消耗2点压力 → 医疗兵不受精神伤害，重装兵承受腐化和压力

### 协作组合范例
```
任务：突破不死者防线
1. 斥候：准备行动（侦查安全路线）→ 改善处境
2. 军官：领导集体行动（组织全队冲锋）→ 领导投组织骰，队员+压力+骰
3. 重装兵：协助军官（掩护侧翼）→ 军官+1d
4. 医疗兵：保护新兵（承担可能的后退后果）→ 新兵安全，医疗兵承受
5. 狙击手：准备行动（压制远程火力）→ 接下来队友效果+1级
```

## 六、效果判定完整指南

### 效果三要素详解

#### 效力（二元判定）
- 针对敌人弱点 = **有效**
- 有额外时间准备 = **有效**
- 愿意冒更大风险 = **有效**
- 使用特殊能力（如武艺） = **有效**
- 没有任何优势 = **无效**

> 效力是门槛：无效时效果等级直接为0或有限。

#### 规模
| 规模等级 | 描述 |
|---------|------|
| 1 | 一两个个体 |
| 2 | 小型队伍（3-5人） |
| 3 | 中型队伍（6-10人） |
| 4 | 大型队伍（20人） |
| 5 | 巨型队伍（40人） |
| 6 | 超巨型队伍（80人） |
| 7 | 极巨型队伍（160人） |

规模差距影响：
- 规模每高1级 → 可恶化处境（危险→绝望）
- 规模每高1级 → 可降低效果1级
- 规模每高1级 → 可提升伤害1级

#### 威胁（装备/生物质量）
| 威胁值 | 对应 |
|--------|------|
| 1 | 普通装备、杂兵 |
| 2 | 优质装备、精英 |
| 3 | 重武器、怖物 |
| 4+ | 攻城武器、副官 |

### 用处境交换效果
在GM声明效果等级后，玩家可以主动提出：将处境**降级**（如危险→绝望）以换取效果**提升**（一般→极佳）。这是一种赌博机制。

## 七、抵抗、压力与创伤

### 抵抗骰机制
当玩家不想接受后果时：
1. 玩家声明要抵抗哪个后果
2. 选择对应属性（认知/技术/决心）投抵抗骰
3. 结果：**6**=完全抵抗，**4-5**=部分抵抗，**1-3**=失败
4. 承受 **6 - 骰子结果** 点**精神压力**

### 精神压力系统（完整版）
- **上限**：6点
- **用途1**：抵抗后果
- **用途2**：超越自我（2点压力 = +1骰 或 +1效果等级 或 失能状态下行动）

#### 压力阈值效果
| 压力值 | 状态 |
|--------|------|
| **0-2点** | 正常，无明显影响 |
| **3-4点** | 感到紧张，部分行动描述应反映焦虑 |
| **5点** | 濒临崩溃，角色可能出现轻微失误、情绪失控 |
| **6点** | **精神创伤**（见下） |

> **AI-GM叙事指引**：当角色压力达到3-4点时，在描述中暗示其紧张状态（手抖、呼吸急促、眼神游离）。5点时，角色可能做出非理性决定。

### 精神创伤
- 压力满6点时 → 受到**精神创伤**（失能状态）
- 选一个**创伤症状**（8种之一）
- 最多2次创伤 = **死亡**
- 扮演创伤 → 获得经验点

#### 压力满6点时的详细流程
1. **清空所有压力标记**（回到0）
2. **获得一次精神创伤**
3. **选择一种创伤症状**（从8种中选择）
4. **角色进入失能状态**（除非使用能力如"不屈"）
5. **如果这是第2次创伤** → 角色**死亡**

#### 压力上限提升
- 默认压力上限为**6点**
- 通过能力**"幸存者"**可提升上限：
  - 第1次选择：上限提升至**8点**
  - 第2次选择：上限提升至**10点**

#### 创伤承受上限提升
- 默认最多承受**2次创伤**（第3次=死亡）
- 通过能力**"老兵"**可提升上限：
  - 第1次选择：上限提升至**3次**
  - 第2次选择：上限提升至**4次**

### 8种创伤症状
冷漠、阴影、痴迷、偏执、鲁莽、软弱、无常、恶毒

## 八、伤害、腐化与死亡

### 伤害等级
| 等级 | 名称 | 范例 | 效果 |
|-----|------|------|------|
| 1级 | 轻微 | 挫伤、擦伤、扭伤、吓到 | 轻度妨碍 |
| 2级 | 中度 | 流血、烧伤、疲劳过度 | 明显妨碍 |
| 3级 | 严重 | 断腿、穿胸、昏厥 | **失能** |
| 4级 | 致命 | 致命毒素、胸部下陷 | **=死亡** |

- **行满溢推**：2级满→升为3级；3级满→升为4级（死亡）
- **护甲抵消**：普通护甲1格，重甲2格，盾牌1格

### 腐化与枯萎病
- 累计 **7点腐化** → 获得1个枯萎症状，枯萎病+1级
- 4级枯萎病 = **变成怪物 = 死亡**
- 腐化**无法**通过一般手段降低
- 不死者造成的腐化通常等于其威胁度

### 8种枯萎症状
咒诅、寄生、饥渴、瘴气、变异、狂怒、腐烂、幻象

## 九、进程表系统

进程表是将复杂局势可视化的核心叙事工具，用扇形格子追踪进度。

### 进程表类型

| 类型 | 用途 | 填满触发 |
|-----|------|---------|
| **危机表** | 敌人疑心增长、仪式进行、气压升高 | 危机爆发 |
| **竞速表** | 两个对立表同时推进 | 谁先满谁赢 |
| **连锁表** | 条件解锁型 | 前置表满后启动 |
| **任务表** | 时间窗口倒计时 | 时间到 |
| **威胁表** | 精英/著名怪物血量 | 怪物被击败 |
| **长期项目** | 军需官大型工程 | 项目完成 |

### 填表规则
- 有限效果 → 1格
- 一般效果 → 2格
- 极佳效果 → 3格
- 进程表代表**障碍**（如"不死者哨兵"），而非手段（如"躲过守卫"）

### 威胁表（怪物血量）
- 威胁度2怪物 → **8格**
- 威胁度3怪物 → **10格**
- 任何合理行动（射击、游击、陷阱、欺骗）都可以填表

## 十、任务类型系统

> **补充说明**：手册v1.0未详细描述任务类型。现补充完整任务分类。

### 标准任务类型

每个地点有一组标准任务，指挥官从中选择主要和次要任务。

| 任务类型 | 描述 | 典型奖励 | 典型惩罚 | 适合专家 |
|---------|------|---------|---------|---------|
| **突袭任务** | 快速打击敌人目标，制造混乱，摧毁设施 | 士气+1，消灭敌人，削弱敌军 | 伤亡，敌人警觉提升，报复 | 重装兵、狙击手、军官 |
| **侦察任务** | 侦查敌情、地形、资源，获取情报 | 情报+1，了解威胁度，发现弱点 | 被发现，遭遇战，情报泄露 | 斥候、军官 |
| **宗教任务** | 与神选者相关的精神/信仰任务，净化、祈祷、守护圣物 | 神选者支持，士气+1，净化腐化 | 亵渎，腐化+，神选者不满 | 书记官、医疗兵 |
| **补给任务** | 获取食物、物资、装备，搜刮废墟，交易 | 食物+，补给+，获得特殊物品 | 物资损失，遭遇伏击，食物污染 | 军需官、军官 |

### 特殊任务
- 每个地点有**3个特殊任务**
- 通常有独特的背景、奖励和惩罚
- 需要消耗**1份情报**，由**GM选择**特殊任务并告知指挥官
- 完成后可能改变战役走向

### 次要任务
- 由指挥官选择的第二个任务
- **不需要玩家扮演执行**，由遭遇骰结果自动决定成败
- 成功 = 获得奖励；失败 = 承受惩罚
- **所有参与了次要任务的专家获得2点xp，不论任务成功与否**
- 可以用情报为次要任务的遭遇骰+1
- 次要任务可以和主要任务派不同的人

### 任务选择流程
```
1. 指挥官查看当前地点的所有可用任务
2. 使用情报提问（了解威胁、敌人、奖励）
3. 选择主要任务（需玩家扮演执行）
4. 选择次要任务（自动结算）
5. 军士长部署人员
6. 投遭遇骰确定初始处境
7. 执行任务
```

### AI-GM任务设计原则
- 主要任务应提供**明确的胜利条件和失败后果**
- 次要任务的成败应影响军团资源
- 特殊任务应与地点特色和破碎者活动相关
- 任务难度应随时间（Time）和压力（Pressure）提升而增加

## 十二、职位系统：指挥官

### 核心职责
- 选择主要任务与次要任务
- 决定进军路线（选择下一个地点）
- 决定是否进军（推进战役）
- **消费情报**获得战略优势

### 核心资源：情报
情报来自任务奖励（主要是侦察任务）。

**情报消费方式**：
1. **遭遇骰+1**（消耗1点情报）：为任何任务增加1个遭遇骰
2. **解锁特殊任务**（消耗情报）：获得执行某地特殊任务的权利
3. **情报问题**（不消耗，提问数量 = 持有的情报数量上限）

### 情报问题列表（按持有量分级）

**0情报**（眼前任务细节）：
- 任务中不死者威胁度最高分别是多少？
- 任务需要多远距离行军？
- 哪件物品带上比较有用？
- 任务中有哪两种可选执行方式？
- 哪些小队不尊重或不信任领导层？
- 神选者对任务的态度是怎样的？

**1情报**（额外问题）：
- 任务中哪些物品比较有用？
- 谁对这个任务很认真？
- 可能遇到什么样的挑战？
- 会遇上哪些破碎者的部队？
- 接下来地点能找到什么有益资源？
- 某个精英的弱点/欲望可能是什么？

**2情报**（战略级）：
- 会遇到哪个副官或臭名昭著的存在？
- 不死者布置中最大的弱点是什么？
- 这个任务和之前的任务有关联吗？
- 破碎者在往哪个方向移动？
- 地点存在什么样的挑战或困难？

**3情报**（战役级）：
- 此前遇到的某个臭名昭著的存在的深层弱点是什么？
- 余烬之王的目标是什么？
- 破碎者如何影响当前地区的？
- 不死者军队正在执行什么计划？
- 军团如何才能最大限度地利用当前局势？

### 按神选者的初始情报
- **施芮亚**：每完成2次主要任务获得1份情报（来自有角者外出探索观察）
- **有角者**：初始情报1
- **左拉**：初始情报0

## 十五、职位系统：军需官

### 核心职责
- 物资调配与补给
- 募兵（补充伤亡）
- 休息与恢复（管理角色状态）
- 长期项目（制造攻城武器、研发药物等）

### 战役行动（每次战役阶段选一项）

#### 1. 获取资源
- 军需官投战役骰
- 结果决定获得多少食物/补给
- 可以通过渠道、勒索等获得加成

#### 2. 放假（R&R）
- 让军团休整
- 每个角色可以勾销伤害格子
- 士气+1
- 但时间和压力会推进

### 休息与恢复详细规则

#### 6小时的休息
- 可以勾销**1格伤害**（任意等级）
- 清除**部分精神压力**（通常2-3点）
- 但如果营地不安全（遭遇骰差），休息效果减半

#### 1天的休息（完整休整）
- 可以勾销**2格伤害**（任意等级）
- 可以**清除所有精神压力**（归零）
- 新兵晋升判定（检查是否存活2次任务）

#### 休息的前提条件
- **必须有足够食物**：每个存活角色消耗1单位食物
- **营地必须相对安全**：由上次遭遇骰和地点特性决定
- **如果食物不足**：无法有效休息，士气下降，角色可能出现饥饿状态

#### 休息时不能做的事
- 休息期间不能执行任务
- 休息期间间谍总管不能派遣间谍执行长期任务（短期任务可以）
- 休息时书记官仍然可以讲故事

#### 3. 长期项目
- 推进大型工程进程表
- 范例：制造攻城武器、研发新药、修理装备
- 通常需要多个战役阶段完成

#### 4. 募兵
- 补充新兵
- 新兵数量取决于军团的声誉和当前的士气
- 新兵默认无特种行动，威胁度1

### 食物消耗与补给系统

> **补充说明**：手册v1.1未提及食物消耗规则。现补充。

#### 食物消耗
- 每次战役阶段（进军后），每个**存活角色**消耗**1单位食物**
- 食物不足时：
  - **士气下降**（-1）
  - **无法有效休息**（休息时只能勾销1格伤害而非2格）
  - **角色可能出现饥饿状态**（GM判断）

#### 初始食物
- 军团起始有**10单位食物**

#### 补给获取途径
1. **补给任务**：完成补给类型任务获得食物
2. **军需官"获取资源"**：投战役骰获得物资
3. **渠道（社交行动）**：利用关系获取额外补给
4. **间谍网络"商人"节点**：强化补给任务额外+1补给
5. **地点搜刮**：某些地点有特殊资源可以搜刮

### 休息与恢复规则
- 每名角色一次任务后可以进行一次休息
- 勾销伤害格子（数量取决于条件）
- 降低精神压力
- 新兵在休息后可以**晋升为士兵**（存活2次任务后）

### 间谍总管联动
军需官执行战役行动时，间谍总管同时派遣间谍。两者在同一阶段进行。

## 十六、职位系统：间谍总管

### 起始配置
- **2位初始间谍**：1位熟练 + 1位大师
- 最多同时拥有 **3位间谍**（需解锁"增员"节点）

### 7位可选间谍

| 间谍 | 特技 | 定位 |
|------|------|------|
| **安托瓦内特** | 起始即为**大师级** | 暗杀/破坏 |
| **鲍提斯** | 扩张网络任务额外+1格 | 导师/建设 |
| **赤烦风** | **任何任务都不会负伤** | 万能安全牌 |
| **伊葛丽德** | 收集情报时**额外多问1个问题** | 情报专家 |
| **莉娅** | 调查长期任务+1骰 | 侦察/潜伏 |
| **昂也廷** | 强化任务长期任务+1骰 | 任务优化 |

### 间谍网络科技树

```
间谍网络（起始）——允许维持2位间谍
├── 增员 → 可同时维持3位间谍
│   └── 训练 → 熟练间谍升为大师（可点2次）
├── 分析师 → 调查长期任务+1骰
│   └── 投资 → 扩张网络长期任务+1骰
├── 现场评估 → 收集情报时额外多问1个问题
├── 诱捕 → 设置陷阱长期任务+1骰
└── 信息源 → 强化任务长期任务+1骰
    ├── 商人 → 强化补给任务额外+1补给
    ├── 教团 → 强化宗教任务额外+1资源
    ├── 佣兵 → 强化突袭任务额外+1士气
    └── 游侠 → 强化侦察任务额外+1情报
```

### 派遣间谍（与军需官同期）

每个战役阶段，一名间谍只能执行**短期任务**或**长期任务**之一。

#### 短期任务（无风险，不投骰）

| 任务 | 效果 |
|------|------|
| **恢复** | 移除间谍的负伤状态 |
| **收集情报** | 从指挥官情报问题列表中选1个问题问GM |
| **勒索** | 为1次"获取资源"战役行动+1骰 |
| **协助** | 为1次"长期项目"战役行动+1骰 |

#### 长期任务（8格进程表，有伤亡风险）

**投骰规则**：
- 熟练间谍：1d
- 大师间谍：2d（取最高）
- 1-3 = 1格（**负伤**）
- 4-5 = 2格
- 6 = 3格
- 双6 = 5格

**伤亡规则**：
- 首次负伤 → 获得负伤状态
- 已负伤再次负伤 → **牺牲（死亡）**
- 负伤间谍执行"恢复"短期任务可清除负伤

| 长期任务 | 完成效果 |
|---------|---------|
| **强化任务** | 指定类型任务奖励与惩罚骰各+1 |
| **扩张网络** | 解锁一个新网络节点 |
| **设置陷阱** | 下轮任务列表必定包含突袭指定副官/怖物的任务 |
| **招募** | 加入1位新间谍（替换死亡者） |
| **调查** | 了解当地所有特殊任务；指挥官已解锁权限时由间谍总管选择执行哪个 |

> 长期任务可接力：不同间谍轮流推进同一任务，暂停无惩罚。

## 二十六、游戏结构：任务+战役双阶段

```
┌─────────────────────────────────────────────────┐
│                    战役阶段                         │
│  ├── 时间流逝（积累时间和压力，消耗食物）            │
│  ├── 各职位执行战役行动                             │
│  │   ├── 军需官：获取资源/长期项目/休息恢复/募兵     │
│  │   ├── 间谍总管：派遣间谍（短期/长期任务）          │
│  │   └── 书记官：讲述军团故事                        │
│  ├── 指挥官决定是否进军                             │
│  └── GM生成新任务 → 进入任务阶段                     │
└──────────────────────┬────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────┐
│                    任务阶段                         │
│  ├── 指挥官选主要任务+次要任务                      │
│  ├── 军士长派人选 + 军需官分配补给                   │
│  ├── 投遭遇骰确定起始处境                            │
│  ├── 主要任务：小队执行，投骰、承担后果              │
│  │   └── 可使用闪回、团队协作                        │
│  ├── 次要任务：由遭遇骰决定结果                      │

```

## 二十六、结算阶段流程

> **补充说明**：手册v1.1未详细描述结算阶段。经PDF审查，现补充完整结算流程。

结算阶段是任务阶段结束后、战役阶段开始前的关键环节，按以下顺序执行：

### 结算顺序

```
1. 主要任务结算
   ├── 存活角色获得任务奖励
   └── 死亡角色从名册移除
2. 次要任务结算
   ├── 投遭遇骰（使用次要任务参与者的调查骰池）
   └── 根据结果分配奖励/惩罚
3. 伤害与压力处理
   ├── 角色勾销伤害（根据休息条件）
   ├── 处理压力满6点的角色（获得创伤）
   └── 处理4级伤害的角色（死亡）
4. 士气调整
   ├── 阵亡：士气-1
   ├── 士气6+：结算时+1士气
   └── 士气3-：结算时-1士气
5. 经验点分配
   ├── 基础XP = 本次最高敌人威胁度
   └── 额外XP（协助/扮演创伤/承受伤害/死亡等）
6. 晋升判定
   └── 新兵存活2次任务 → 晋升为士兵
7. 进入"在营地"场景
```

### 主要任务结算
- 所有参与主要任务且**存活**的角色获得任务奖励（通常是士气+1或物资）
- **死亡**的角色从名册中移除（但保留在编年史中）
- 每个死亡角色使**士气-1**

### 次要任务结算
- 次要任务**自动结算**，无需玩家扮演执行过程
- 使用**次要任务参与者的调查骰池**投遭遇骰
- 结果对应地点的次要任务奖励/惩罚表：
  - **1** = 惩罚A（通常是士气-1或物资损失）
  - **2** = 惩罚B
  - **3** = 惩罚C
  - **4** = 奖励A（通常是情报或物资）
  - **5** = 奖励B
  - **6** = 奖励C（通常是士气+1或稀有物资）

> **AI-GM提示**：每个地点应在战役准备阶段预设次要任务表。次要任务可以和主要任务派不同的人。

### 伤害与压力处理
- 角色在本次任务中承受的伤害和压力保留到结算
- **压力满6点**：清空压力→获得精神创伤→选择症状→失能
- **4级伤害**：角色死亡
- 如果有角色同时满足压力满和4级伤害，先处理死亡

### 士气调整
| 条件 | 士气变化 |
|------|---------|
| 有角色阵亡 | -1 |
| 士气当前≥6 | +1（庆祝胜利） |
| 士气当前≤3 | -1（恐慌蔓延） |
| 完成宗教任务 | +1 |
| 完成突袭任务 | +1 |

### 晋升判定
- 每个新兵检查**存活任务次数**
- 达到**2次存活** → 晋升为士兵
- 晋升时：获得忍耐特种行动1级，获得1个新能力

## 二十七、在营地场景

> **补充说明**：手册v1.1提到书记官讲故事，但没有完整的"在营地"场景规则。现补充。

"在营地"是每次任务结束后、战役阶段开始前的叙事场景。根据当前**士气水平**，场景氛围截然不同。

### 士气0-3：沮丧营地

**氛围**：沉默、易怒、绝望、阴影笼罩

**场景特征**：
- 角色们围坐在微弱的篝火旁，很少有人说话
- 有人低声咒骂，有人盯着火焰发呆
- 书记官试图讲故事，但声音被风声吞没
- 可能听到远处的啜泣声

**机制效果**：
- 书记官讲故事时，听众很难获得经验点（需要极好的角色扮演）
- 休息时只能勾销**1格伤害**（而非正常的2格）
- **士气3或更低**：结算时额外-1士气
- 可能出现**逃兵事件**（GM可引入叛逃剧情）

**AI-GM叙事建议**：
- 描述潮湿、寒冷、腐臭味
- 让玩家角色感受到"我们可能要死在这里"
- 如果有人试图鼓舞士气，让其他人报以冷漠或嘲讽

### 士气4-7：普通营地

**氛围**：疲惫但仍在坚持、例行公事、麻木

**场景特征**：
- 角色们在做例行维护：磨刀、检查装备、包扎伤口
- 有人低声交谈，有人默默进食
- 书记官讲述军团故事，听众心不在焉但仍在听
- 医疗兵在帐篷间穿梭，处理伤员

**机制效果**：
- 书记官讲故事正常结算经验点
- 休息时可以正常勾销伤害和清除压力
- 标准的食物消耗和补给分配

**AI-GM叙事建议**：
- 描述日常军事生活的细节
- 让玩家感受到"这只是又一个难熬的夜晚"
- 鼓励玩家角色之间的小互动（分享口粮、交换战利品）

### 士气8-10：振奋营地

**氛围**：士气高涨、歌声、笑声、充满希望

**场景特征**：
- 篝火旁有人唱歌，有人讲笑话
- 角色们分享战利品和故事
- 书记官的故事引起共鸣，笑声和掌声
- 可能有新兵主动要求加入军团

**机制效果**：
- 书记官讲故事时，所有参与者额外+1经验点
- 休息时额外勾销**1格伤害**
- **士气6+**：结算时额外+1士气
- 可能出现**自愿加入的新兵**（军需官募兵+1）

**AI-GM叙事建议**：
- 描述温暖、安全感（相对的）、战友情谊
- 让玩家感受到"我们也许能活下去"
- 注意：不要太过乐观——这只是暴风雨前的宁静

### 营地活动（所有士气等级）

在营地场景中，玩家角色可以进行以下活动：

| 活动 | 效果 | 限制 |
|------|------|------|
| **休息** | 勾销伤害，清除压力 | 需要安全营地和食物 |
| **修理装备** | 修补行动恢复装备 | 需要修补工具和材料 |
| **交换情报** | 社交行动分享信息 | 无限制 |
| **祷告/冥想** | 动摇行动恢复精神 | 需要安静环境 |
| **训练新兵** | 训诫行动帮助新兵 | 需要老兵参与 |
| **写遗书** | 无机制效果，纯叙事 | 任何角色 |

## 二十八、游戏结构：任务+战役双阶段

### 核心叙事原则

1. **部分成功驱动叙事**：4-5是最常见结果 → 玩家总是付出代价 → 符合军事撤退的绝望感
2. **精神压力=英雄主义的燃料**：超越自我让人做到不可能的事，但代价是崩溃 → 英雄与疯狂一体两面
3. **角色轮换降低死亡恐惧**：角色死了换下一个 → 敢死队氛围
4. **双阶段结构**：战术小队行动 + 战略层管理 → 既有紧张感又有深度
5. **恶魔的交易=道德困境**：玩家主动提出彼此代价 → 团队内自然发生背叛/牺牲
6. **闪回=创造性解决**：鼓励玩家用闪回创造有利条件，但代价是压力
7. **团队协作=集体牺牲**：鼓励玩家互相协助，但协助也要付出代价

### GM叙事建议

**开场与氛围**
- 强调泥、血、雨、黑暗
- 军团不是在"冒险"，是在"求生"
- 每个决定都有代价

**描述敌人**
- 不死者是恐怖的存在，不只是"怪物"
- 描述腐化的视觉细节：肿胀、变色、不自然的肢体
- 破碎者的出现应该让玩家感到真正的恐惧

**处理死亡**
- 角色死亡时给予告别时刻
- 让书记官记录他们的牺牲
- 新兵接替时强调"又一张新面孔"

**推进腐化**
- 逐渐描述角色的枯萎症状
- 让其他角色注意到这些变化
- 制造"要不要抛弃被腐化的战友"的困境

**使用进程表**
- 让玩家看到危机表逐渐填满的紧张感
- 用竞速表制造"争分夺秒"的紧迫感
- 威胁表填满时给击败精英的成就感

**闪回引导**
- 当玩家陷入困境时主动提示"你可以闪回到..."
- 要求闪回的细节必须合理
- 闪回消耗压力，保持机制平衡

**团队协作引导**
- 鼓励玩家描述如何协助彼此
- 集体行动时让领导者承担更多风险
- 准备行动应该被经常使用

### AI-GM判定速查

| 情况 | 判定 |
|------|------|
| 玩家宣布行动 | 六步法：目标→行动→处境→效果→奖励→投骰 |
| 玩家不想承受后果 | 投抵抗骰（认知/技术/决心），承受6-结果点压力 |
| 压力满6点 | 精神创伤，选8种症状之一 |
| 压力5点 | 濒临崩溃，角色可能出现轻微失误 |
| 压力3-4点 | 感到紧张，描述应反映焦虑 |
| 伤害行满 | 向上溢推，3级满→4级=死亡 |
| 腐化满7点 | 获得枯萎症状，枯萎病+1级 |
| 遭遇骰 | 1-3绝望/4-5危险/6安全 |
| 敌人威胁度高 | 效果降1级/伤害升1级/腐化=威胁度 |
| 任务完成 | XP = 最高威胁度 |
| 玩家陷入困境 | 提示闪回（消耗1-2压力） |
| 多人协作 | 协助+1d / 领导集体+最多3d / 准备改善处境 / 保护承受后果 |

| 条件 | XP奖励 |
|------|--------|

### 经验表填满规则
- **6点XP**填满一个经验表（属性/特种/角色卡）
- 填满后擦净所有格子，获得**1次提升**
- 提升选项：
  - 选择**一项新能力**
  - 或将**某项行动等级提升1级**
  - 或将**某项特种行动等级提升1级**
- 溢出XP保留，填入下一个经验表

### 闪回速查
| 复杂度 | 压力消耗 | 范例 |
|--------|---------|------|
| 简单 | 1点 | 提前藏好物品、偷了钥匙 |
| 中等 | 2点 | 提前侦查路线、建立关系 |
| 复杂 | 2点+后果 | 提前设置陷阱被敌人发现 |

### 团队协作速查
| 类型 | 消耗 | 效果 | 风险 |
|------|------|------|------|
| 协助 | 1压力 | 被协助者+1d | 协助者可能承受后果 |
