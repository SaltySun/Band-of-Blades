## 文档库使用规范（给AI的提示）

1. **运行时优先L3**：主持游戏时，遇到规则问题先查L3对应卡片，确保速度和准确性。
2. **L3不够时上L2**：如果L3卡片没有涵盖特殊情况，去L2查详细规则。
3. **冲突时回L1**：如果不同文件对同一规则的描述不一致，以 `L1_Raw/刀锋连队_AI-GM手册_v1.3.md` 为准。
4. **不要依赖记忆**：即使你已经"知道"规则，涉及具体数字时（如威胁表血量、XP数值、食物消耗），也请查阅文档确认。
5. **交叉验证**：涉及角色生死、重大奖励/惩罚的判定，建议同时查阅L2和L3确保准确。
