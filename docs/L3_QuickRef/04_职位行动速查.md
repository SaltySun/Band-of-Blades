# 职位行动速查

> **场景**：战役阶段，某职位玩家要执行行动时

---

## 指挥官

```
[行动] 选择主要任务 + 次要任务
[行动] 决定进军路线（选下一个地点）
[行动] 决定是否进军（推进战役）
[资源] 情报（消费：遭遇骰+1 / 解锁特殊任务 / 提问）
```

**情报问题分级（持有量=可问等级）：**
- 0情报：任务细节（威胁度、距离、物品、执行方式、态度）
- 1情报：额外问题（有用物品、认真的人、挑战、破碎者部队、资源、弱点）
- 2情报：战略级（副官、弱点、关联、移动方向、困难）
- 3情报：战役级（深层弱点、余烬之王目标、破碎者影响、不死者计划、利用局势）

---

## 军士长

```
[行动] 部署人员（主要/次要任务）
[行动] 投遭遇骰（1-3绝望 / 4-5危险 / 6安全）
[资源] 士气（0-10）
```

**派遣原则：**
- 失能/创伤/高腐化不宜派遣
- 能力匹配任务类型
- 一角色不能同时参与主要和次要任务

**士气调整：**
- 阵亡-1 / ≥6时+1 / ≤3时-1 / 宗教任务+1 / 突袭+1

---

## 书记官

```
[行动] 讲述军团故事（每次战役阶段一次）
[效果] 听众获得对应属性经验点
[加成] 士气8-10时，听众额外+1 XP
```

**编年史类型：**
1. 军团建立的故事
2. 军团独立的故事
3. 战火淬炼的故事
4. 意志不屈的故事
5. 军团存在的意义

---

## 军需官（每次战役阶段选一项）

| 行动 | 机制 | 效果 |
|------|------|------|
| **获取资源** | 投战役骰 | 获得食物/补给 |
| **放假（R&R）** | — | 士气+1，勾销伤害，但时间压力推进 |
| **长期项目** | 推进进程表 | 制造武器/研发药物等 |
| **募兵** | — | 补充新兵（取决于声誉和士气） |

**休息规则：**
- 6小时：勾销1格伤害，清除2-3点压力
- 1天：勾销2格伤害，清空压力，新兵晋升判定
- 前提：足够食物 + 安全营地

**食物：**
- 初始10单位
- 每次战役阶段：存活角色每人消耗1单位
- 不足：士气-1，休息效果减半

---

## 间谍总管（与军需官同期执行）

**配置：** 2位初始间谍（1熟练+1大师），最多3位

### 短期任务（无风险，不投骰）

| 任务 | 效果 |
|------|------|
| **恢复** | 移除负伤状态 |
| **收集情报** | 问指挥官情报问题列表中的1个问题 |
| **勒索** | 为军需官"获取资源"+1骰 |
| **协助** | 为军需官"长期项目"+1骰 |

### 长期任务（8格进程表，有伤亡风险）

**投骰：**
- 熟练间谍：1d
- 大师间谍：2d取高
- 1-3=1格（负伤）/ 4-5=2格 / 6=3格 / 双6=5格

**伤亡：** 首次负伤→负伤状态 / 已负伤再负伤→死亡

| 任务 | 完成效果 |
|------|---------|
| **强化任务** | 指定类型任务奖励与惩罚骰各+1 |
| **扩张网络** | 解锁一个新网络节点 |
| **设置陷阱** | 下轮任务列表必定包含突袭指定副官/怖物的任务 |
| **招募** | 加入1位新间谍 |
| **调查** | 了解当地所有特殊任务 |

**间谍网络科技树：**
- 增员→训练 / 分析师→投资 / 现场评估 / 诱捕 / 信息源→商人/教团/佣兵/游侠
