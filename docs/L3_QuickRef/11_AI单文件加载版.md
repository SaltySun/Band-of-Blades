# AI单文件加载版

> **用途**：当AI上下文窗口有限时，只加载这一个文件即可运行基础游戏。
> **设计原则**：极度压缩，只保留不可推导的核心数字和流程。
> **体积**：约5KB，可一次性加载。

---

## 核心数字（不可错）

| 项目 | 数值 |
|------|------|
| 骰子 | d6，取最大，6=成功，4-5=部分成功+1后果，1-3=失败+1-2后果，双6+=关键成功 |
| 骰池≤0 | 投2d取较小值 |
| 奖励骰上限 | +2d |
| 压力上限 | 6（幸存者→8→10） |
| 创伤上限 | 2（老兵→3→4） |
| 压力满6 | 清空压力→获得创伤→失能 |
| 伤害 | 1轻/2中/3重失能/4死；行满向上溢推 |
| 腐化满7 | 枯萎症状，枯萎病+1；4级枯萎=死 |
| 士气 | 0-10；0-3沮丧/4-7普通/8-10振奋 |
| 食物 | 初始10；每次战役阶段存活角色各消耗1 |
| 经验表 | 6格满→行动等级+1；上限3（精英→4） |
| 新兵晋升 | 存活2次任务→士兵 |
| 次要任务XP | 参与专家+2XP（不论成败） |
| 基础XP | =最高敌人威胁度 |
| 威胁表 | 威胁度2→8格/3→10格/4→12格 |
| 遭遇骰 | 1-3绝望/4-5危险/6安全 |
| 完整战役 | 约12节 |

## 六步法（严格顺序）

```
1.玩家宣布目标
2.玩家选行动（11种：调查射击修补社交侦查训诫游击破坏机动组织动摇）
3.GM设定处境（安全/危险默认/绝望+1经验）
4.GM设定效果（无效0/有限1/一般2/极佳3/极限4+）
5.玩家加奖励骰（协助+1d耗1压/超越+1d或+1效果耗2压/恶魔交易+1d）
6.投骰判断
```

## 抵抗骰

```
不想承受后果→投对应属性抵抗骰→承受压力=6-结果
6=完全抵抗/4-5=部分/1-3=失败
```

## 四种协作

| 类型 | 消耗 | 效果 | 风险 |
|------|------|------|------|
| 协助 | 1压 | +1d | 协助者可能承受后果 |
| 领导集体 | 参与者各1压 | 领导+最多3d | 失败全员承受 |
| 准备 | 通常无 | 改善处境或+1效果 | 失败承受后果 |
| 保护 | 1-2压 | 代替队友承受 | 保护者承受全部 |

## 闪回

```
玩家卡壳时提示闪回
简单=1压/复杂=2压
不能改变已确定事实
闪回中行动仍需投骰
```

## 结算七步（不可颠倒）

```
①主要任务结算（死亡移除，士气-1/人）
②次要任务（调查骰投遭遇骰，1-3惩罚/4-6奖励）
③伤害压力处理（满6→创伤，4级→死亡；同时满足先处理死亡）
④士气调整（阵亡-1/≥6+1/≤3-1）
⑤XP分配（基础=最高威胁度，专家次要+2）
⑥晋升（新兵2次存活→士兵）
⑦营地（按士气0-3沮丧/4-7普通/8-10振奋）
```

## 营地休息

```
6小时：勾销1格伤，清2-3压
1天：勾销2格伤，清空压力
前提：食物充足+营地安全
沮丧营地：效果减半
振奋营地：额外+1格
```

## 职位速查

**指挥官**：选主/次要任务，消费情报（遭遇+1/解锁特殊任务/提问）
**军士长**：部署人员，投遭遇骰，管理士气0-10
**军需官**：四选一行动（获取资源/放假+1士气/长期项目/募兵）
**书记官**：讲故事→听众+XP；士气8-10时额外+1XP
**间谍总管**：派间谍（短期无风险/长期8格进程表有伤亡）

## 神选者

- 三选一：施芮亚（军事/情报）、有角者（变形/灵活）、左拉（正面/强力）
- 每次任务一次能力，免疫腐化，可执行超凡使命
- 最终战：只有神选者能伤害破碎者

## 常见错误提醒

1. 效果等级≠骰子结果（独立判定）
2. 敌人威胁度高→你的效果-1/伤害+1（不是反过来）
3. 压力满6=创伤（不是死亡），第2次创伤=死
4. 次要任务专家+2XP不论成败
5. 闪回必须消耗1-2压力
6. 结算顺序不可颠倒（死亡处理在士气和XP之前）
7. 进程表=障碍（任何合理行动可填），威胁表=敌人血量（攻击填）
