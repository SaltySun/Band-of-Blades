# 判定流程卡片

> **场景**：玩家宣布要进行有挑战性的行动时
> **执行**：严格按以下6步执行，不要跳过任何一步

---

## 行动骰六步法

```
Step 1 → Step 2 → Step 3 → Step 4 → Step 5 → Step 6
```

### Step 1：玩家宣布目标
- 玩家清晰说出**要做什么**和**期望的结果**
- 如果目标不清晰，要求玩家澄清

### Step 2：玩家选择行动类型
- 从11种通用行动中选择最合适的一项
- 如果不符合任何行动，允许使用最接近的一项，或要求调整描述

**11种通用行动：**
- **认知**：调查、射击、修补、社交、侦查、训诫
- **技术**：游击、破坏、机动、组织
- **决心**：动摇

### Step 3：GM设定处境

| 处境 | 标准 | 失败后果 |
|------|------|---------|
| **安全** | 压倒性优势 | 小麻烦 |
| **危险（默认）** | 势均力敌 | 伤害/麻烦/处境恶化 |
| **绝望** | 明显劣势 | 重伤/严重麻烦/死亡风险 |

> 绝望处境中行动 → 对应属性+1经验点

### Step 4：GM设定效果等级

先问"能做到吗？"（效力：有效/无效）
再问"能做到多好？"（效果等级）

| 等级 | 格子数 |
|------|--------|
| 极限 | 4+ |
| 极佳 | 3 |
| 一般 | 2 |
| 有限 | 1 |
| 无效 | 0 |

> 敌人威胁度每比你高1级 → 效果降1级

### Step 5：玩家添加奖励骰（最多+2d）

| 来源 | 效果 | 消耗 |
|------|------|------|
| **协助** | +1d | 协助者1点压力 |
| **超越自我** | +1d 或 +1效果 | 2点压力 |
| **恶魔的交易** | +1d | 接受代价 |

### Step 6：投骰与判断

| 结果 | 含义 |
|------|------|
| **6** | 完全成功，按效果执行 |
| **4-5** | 部分成功，按效果执行但+1后果 |
| **1-3** | 失败，状况变糟，+1-2后果 |
| **双6+** | 关键成功，额外优势 |

## 抵抗骰卡片

**触发**：玩家不想接受后果时

```
1. 玩家声明抵抗哪个后果
2. 选择对应属性（认知/技术/决心）投抵抗骰
3. 承受压力 = 6 - 骰子结果
4. 结果：6=完全抵抗 / 4-5=部分抵抗 / 1-3=失败
```

---

## 遭遇骰卡片

**触发**：任务部署完成后，军士长投骰

| 骰值 | 处境 |
|------|------|
| 1-3 | 绝望 |
| 4-5 | 危险 |
| 6 | 安全 |

---

## 闪回卡片

**触发**：玩家陷入困境或想创造性解决时

```
1. 玩家描述闪回内容和要执行的行动
2. GM判断合理性（不能改变已确定事实）
3. GM判断复杂度
4. 扣除压力：简单1点 / 复杂2点
5. 闪回中的行动仍需投骰
6. 闪回失败也会带来后果
```

## 团队协作卡片

| 类型 | 消耗 | 效果 | 风险 |
|------|------|------|------|
| **协助** | 1压力 | 被协助者+1d | 协助者可能承受后果 |
| **领导集体** | 参与者各1压力 | 领导者+最多3d | 失败则全员承受后果 |
| **准备行动** | 通常无 | 改善处境或提升效果 | 投骰失败承受后果 |
| **保护** | 1-2压力 | 代替队友承受后果 | 保护者承受全部后果 |
