# 数据格式规范

> **场景**：需要读取/存储游戏状态、生成导出数据时

---

## localStorage 自动保存键

| 键名 | 内容 |
|------|------|
| `blades-gm-players` | 玩家信息 |
| `blades-gm-chars` | 角色数据 |
| `blades-gm-legion` | 军团状态 |
| `blades-gm-deploy` | 部署记录 |

## 关键字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `actions` | Object | 11个行动的等级和经验，键名使用英文小写 |
| `harm` | Object | 4级伤害，每行有filled/max |
| `stressMax` | Number | 默认6，"幸存者"可提升至8/10 |
| `traumaMax` | Number | 默认2，"老兵"可提升至3/4 |
| `missionsSurvived` | Number | 新兵用，满2次晋升 |
| `rotLevel` | Number | 枯萎病等级，0-4，4=死亡 |
| `isRetired` | Boolean | 角色是否退役（未被选中参与任务） |
