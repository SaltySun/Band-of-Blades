# L3 AI-GM速查层 — 使用指南

> **目标读者**：运行刀锋连队战役的AI代理
> **设计原则**：极简、准确、零歧义。每个文件都是一张"卡片"，用于特定场景的快速决策。

---

## 三层文档库结构

```
docs/
├── L1_Raw/           ← 原始素材（完整文档，用于查证）
├── L2_Structured/    ← 结构化规则（按主题组织的知识）
└── L3_QuickRef/      ← 你在这里（运行时速查卡片）
```

## 查阅优先级

**运行时遇到规则问题 → 按以下顺序查阅：**

1. **L3 速查卡片**（本目录）
   - 找对应的场景卡片
   - 如果卡片内容不够，去L2查详细规则

2. **L2 结构化规则**（`../L2_Structured/`）
   - `00_索引与交叉引用.md` → 定位主题
   - 对应主题文件 → 获取完整规则

3. **L1 原始素材**（`../L1_Raw/`）
   - 当L2/L3出现冲突或需要原文时
   - 以 `刀锋连队_AI-GM手册_v1.3.md` 为最高权威
