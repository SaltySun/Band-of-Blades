# 经验与升级速查

> **场景**：任务结束，需要分配XP和判定升级时

---

## XP来源速查

| 条件 | XP |
|------|-----|
| 基础XP | = 本次最高敌人威胁度 |
| 次要任务（专家） | +2（不论成败） |
| 狙击手射杀威胁度≥2 | +1 |
| 危险/绝望处境行动 | 对应属性+1 |
| 成功协助他人 | +1 |
| 营地有趣扮演 | +1（GM判断） |
| 扮演创伤症状制造困难 | +1 |
| 承受精神创伤 | +1 |
| 承受3级以上伤害 | +1 |
| 角色死亡（遗赠） | +1（给新角色） |
| 士气8-10时书记官讲故事 | 听众额外+1 |
| 新兵晋升士兵 | +1 |

## 升级规则

- 填满某行动的**经验表**（通常6格）→ 该行动等级+1
- 等级上限默认**3级**
- "精英"能力可提升至**4级**

## 晋升规则

| 类型 | 条件 | 结果 |
|------|------|------|
| 新兵→士兵 | 存活2次任务 | 获得忍耐1级 + 1个新能力 |
