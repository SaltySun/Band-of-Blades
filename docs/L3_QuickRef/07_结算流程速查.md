# 结算流程速查

> **场景**：任务阶段结束，进入结算时
> **执行顺序：严格按以下顺序，不要颠倒**

---

## 结算七步

```
① 主要任务结算
   → 存活角色获奖励
   → 死亡角色移除（士气-1/人）

② 次要任务结算
   → 用参与者调查骰池投遭遇骰
   → 1-2-3=惩罚A/B/C / 4-5-6=奖励A/B/C

③ 伤害与压力处理
   → 勾销伤害（根据休息条件）
   → 压力满6→创伤（选症状，失能）
   → 4级伤害→死亡
   → 同时满足：先处理死亡

④ 士气调整
   → 阵亡-1
   → 士气≥6→+1
   → 士气≤3→-1

⑤ 经验点分配
   → 基础XP=最高敌人威胁度
   → 次要任务专家+2XP
   → 其他条件见《06_经验与升级速查.md》

⑥ 晋升判定
   → 新兵存活2次任务→晋升士兵
   → 晋升：忍耐1级+1能力

⑦ 进入"在营地"场景
   → 根据士气水平描述氛围
   → 进行营地活动
   → 准备进入下一战役阶段
```

## 营地氛围速查

| 士气 | 氛围 | 休息效果 | 特殊事件 |
|------|------|---------|---------|
| 0-3 | 沮丧 | 减半 | 可能逃兵，结算-1 |
| 4-7 | 普通 | 正常 | — |
| 8-10 | 振奋 | 额外+1格 | 可能募兵，结算+1 |
