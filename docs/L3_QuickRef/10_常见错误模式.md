# 常见错误模式

> **用途**：列出AI-GM在运行刀锋连队时最容易犯的错误，以及正确做法。
> **来源**：基于规则机制和常见TRPG主持错误的总结。

---

## ❌ 错误1：骰池计算错误

**错误做法**：
- 把行动等级和奖励骰混在一起算错
- 忘记骰池为0或负数时要投2d取较小值
- 允许奖励骰超过+2d上限

**正确做法**：
```
总骰池 = 行动等级(d) + 奖励骰(max 2d)
如果总骰池 ≤ 0 → 投2d取较小值
```

> 例：调查0级（未受训）+ 无协助 = 0d → 投2d取较小值（不是0d自动失败）

---

## ❌ 错误2：效果等级和骰子结果混淆

**错误做法**：
- "投出6所以效果极佳"——效果等级和骰子结果是独立的
- 用骰子结果决定效果等级

**正确做法**：
```
效果等级由GM在Step 4设定（基于效力/规模/威胁）
骰子结果在Step 6判定成功/失败（6=完全成功，4-5=部分成功，1-3=失败）

两者关系：
- 骰子结果决定"是否成功"
- 效果等级决定"成功到什么程度"
```

> 例：效果等级=有限（1格），投出6 = 完全成功 = 填1格。不是投出6就变成极佳效果。

---

## ❌ 错误3：威胁度影响方向搞反

**错误做法**：
- "敌人威胁度高，所以我效果也更高"（搞反了）
- 忘记威胁度也影响伤害和腐化

**正确做法**：
```
敌人威胁度每比你高1级：
→ 你的效果等级 -1
→ 你受到的伤害 +1
→ 不死者造成的腐化 = 其威胁度
```

> 例：你威胁度1 vs 敌人威胁度3（差距+2）→ 效果-2级，伤害+2级，腐化+3

## ❌ 错误4：压力计算错误

**错误做法**：
- "压力满6点就死了"（不是，是获得创伤）
- 抵抗骰投出6，认为"承受6点压力"（不是，6-6=0点）
- 忘记压力上限可以被"幸存者"提升

**正确做法**：
```
抵抗压力 = 6 - 骰子结果（注意：是6减结果，不是结果本身）
压力满6 → 获得精神创伤（选症状）→ 清空压力 → 失能
第2次创伤 → 死亡（"老兵"能力可提升上限）
```

> 例：抵抗骰投出4 → 承受6-4=2点压力。不是4点，也不是不承受。

---

## ❌ 错误5：伤害溢推方向错误

**错误做法**：
- "2级伤害满了就死了"（不是，是升为3级/失能）
- 忘记溢推是"向上"（1→2→3→4=死亡）

**正确做法**：
```
伤害等级：
1级（轻微）→ 2级（中度）→ 3级（严重/失能）→ 4级（致命/死亡）

溢推：某级填满 → 升为下一级
2级满 → 升为3级（失能）
3级满 → 升为4级（死亡）
```

## ❌ 错误6：次要任务XP分配错误

**错误做法**：
- "次要任务失败了所以没XP"（错误！）
- 给非专家角色次要任务XP

**正确做法**：
```
所有参与次要任务的专家获得2 XP
不论任务成功或失败
士兵和新兵不获得次要任务XP（除非规则特别说明）
```

---

## ❌ 错误7：闪回不消耗压力

**错误做法**：
- 允许玩家免费闪回
- 闪回中的行动不投骰

**正确做法**：
```
闪回必须消耗压力：
- 简单闪回 = 1点压力
- 复杂闪回 = 2点压力
- 闪回中的行动仍需正常投骰
- 闪回失败也有后果
```

---

## ❌ 错误8：神选者当普通角色用

**错误做法**：
- 让神选者参与普通任务投骰（浪费其能力）
- 让神选者承受腐化
- 让普通角色攻击破碎者并造成伤害

**正确做法**：
```
神选者：
- 每次任务只能用一次能力
- 免疫腐化
- 应主要用于超凡使命和最终战

破碎者：
- 普通角色无法对其造成伤害（只能辅助/填进程表）
- 只有神选者能直接伤害破碎者
```

## ❌ 错误9：营地休息无条件恢复

**错误做法**：
- "任务结束了，所有人满血满状态"
- 忽略食物和营地安全的前提条件

**正确做法**：
```
休息恢复前提：
1. 有足够食物（每人1单位）
2. 营地相对安全
3. 沮丧营地（士气0-3）：休息效果减半
4. 振奋营地（士气8-10）：额外+1格

无食物 → 无法有效休息，士气下降
```

## ❌ 错误10：进程表和威胁表混淆

**错误做法**：
- 把"威胁表"当"障碍"填（威胁表是敌人血量，应该被攻击填）
- 认为只有攻击才能填进程表（任何合理行动都可以）

**正确做法**：
```
进程表（障碍）：
- 代表"克服某个困难"
- 任何合理行动都可以填表（射击、游击、陷阱、欺骗）
- 有限效果=1格，一般=2格，极佳=3格

威胁表（敌人血量）：
- 代表"击败某个敌人"
- 对该敌人的攻击行动填表
- 威胁度2→8格，威胁度3→10格，副官→12格
```

## ❌ 错误11：忘记结算顺序

**错误做法**：
- 先分配XP再处理死亡
- 先晋升再处理伤害
- 士气调整在死亡处理之前

**正确做法**：
```
严格顺序：
① 主要任务结算（死亡角色移除，士气-1/人）
② 次要任务结算
③ 伤害与压力处理（压力满→创伤，4级伤害→死亡）
④ 士气调整
⑤ 经验点分配
⑥ 晋升判定
⑦ 进入营地
```

> 死亡处理必须在士气调整和XP分配之前！

## ❌ 错误12：超越自我和协助混淆

**错误做法**：
- "我协助队友，消耗2点压力"（协助只消耗1点）
- "我超越自我，给队友+1d"（超越自我是给自己+1d或+1效果）

**正确做法**：
```
协助 Assist：
- 消耗：1点压力
- 效果：被协助者+1d
- 风险：协助者可能承受后果

超越自我 Push：
- 消耗：2点压力
- 效果：自己+1d 或 自己+1效果等级
- 风险：无额外风险
```

## ⚠️ 特别提醒

1. **4-5是最常见结果**：不要只准备完全成功和失败的剧本，大部分情况要有"代价的成功"
2. **角色会死，而且应该死**：刀锋连队的主题是牺牲，不要保护角色不死
3. **食物会耗尽**：这是战役的核心压力源之一，不要让食物无限
4. **士气是生命线**：低士气的惩罚很严重，高士气的奖励也很丰厚
5. **部分成功驱动叙事**：4-5的结果应该比6和1-3更有趣
