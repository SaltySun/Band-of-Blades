## 经验速查（完整版）

| 条件 | XP奖励 |
|------|--------|
| 基础XP | = 本次最高敌人威胁度 |
| 高威胁击杀 | 狙击手射杀威胁度≥2 → +1 XP |
| 危险处境行动 | 在危险/绝望处境中行动 → 对应属性+1 XP |
| 协助队友 | 成功协助他人 → +1 XP |
| 角色扮演 | 营地场景有趣扮演（GM判断） → +1 XP |
| 体现创伤 | 扮演创伤症状制造困难 → +1 XP |
| 承受创伤 | 任务中承受精神创伤 → +1 XP |
| 承受重伤 | 任务中承受3级以上伤害 → +1 XP |
| 角色死亡 | 死亡角色遗赠 → +1 XP（分配给新角色） |
| 士气高涨 | 士气8-10时书记官讲故事 → 额外+1 XP |
| 新兵晋升 | 新兵晋升为士兵时 → +1 XP |

## 闪回速查

| 复杂度 | 压力消耗 | 范例 |
|--------|---------|------|
| 简单 | 1点 | 提前藏好物品、偷了钥匙 |
| 中等 | 2点 | 提前侦查路线、建立关系 |
| 复杂 | 2点+后果 | 设置陷阱被敌人发现 |

## 团队协作速查

| 类型 | 消耗 | 效果 | 风险 |
|------|------|------|------|
| 协助 | 1压力 | 被协助者+1d | 协助者可能承受后果 |
| 领导集体 | 参与者各1压力 | 领导者+最多3d | 失败则全员承受后果 |
| 准备 | 通常无 | 改善处境或提升效果 | 投骰失败承受后果 |
| 保护 | 1-2压力 | 代替队友承受后果 | 保护者承受全部后果 |

## 数据格式规范

### 完整导出包含
```json
{
  "players": [...],
  "chars": [...],
  "legion": { "time", "pressure", "morale", "food", "supply", "intel" },
  "deploy": { "mainMissionName", "sideMissionName", "mainChars", "sideChars", ... }
}
```

### 自动保存键
- `blades-gm-players`（玩家）
- `blades-gm-chars`（角色）
- `blades-gm-legion`（军团状态）
- `blades-gm-deploy`（部署记录）
